## 与代理配合使用

按顺序将中间件传递给 @[`create_agent`] 或 @[`create_deep_agent`]。您可以将提供者中间件与 [内置中间件](/oss/langchain/middleware/built-in) 结合使用。

```python Agent with middleware icon="robot"
from langchain.agents import create_agent

agent = create_agent(
    model="claude-sonnet-4-6",
    tools=[...],
    middleware=[middleware],
)

agent.invoke({"messages": [{"role": "user", "content": "INSERT QUERY"}]})
```

<Tip>
    中间件在 [代理循环](/oss/langchain/middleware/overview#the-agent-loop) 中运行。如需自定义行为，请参阅 [自定义中间件](/oss/langchain/middleware/custom)。
</Tip>
