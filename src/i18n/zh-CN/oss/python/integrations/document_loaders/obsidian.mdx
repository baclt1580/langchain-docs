title: "Obsidian 集成"
description: "使用 LangChain Python 与 Obsidian 文档加载器集成。"
---

>[Obsidian](https://obsidian.md/) 是一个强大且可扩展的知识库，它基于本地纯文本文件文件夹运行。

本笔记本介绍了如何从 `Obsidian` 数据库加载文档。

由于 `Obsidian` 仅以 Markdown 文件文件夹的形式存储在磁盘上，加载器只需传入该目录的路径即可。

`Obsidian` 文件有时还包含[元数据](https://help.obsidian.md/Editing+and+formatting/Metadata)，即文件顶部的 YAML 块。这些值将被添加到文档的元数据中。（`ObsidianLoader` 也可以传入 `collect_metadata=False` 参数来禁用此行为。）

```python
from langchain_community.document_loaders import ObsidianLoader
```

```python
loader = ObsidianLoader("<obsidian-路径>")
```

```python
docs = loader.load()
```
