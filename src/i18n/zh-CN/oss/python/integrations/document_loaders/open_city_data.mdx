[Socrata](https://dev.socrata.com/foundry/data.sfgov.org/vw6y-z8j6) 为城市开放数据提供了 API。

对于像 [SF 犯罪数据](https://data.sfgov.org/Public-Safety/Police-Department-Incident-Reports-Historical-2003/tmnf-yvry) 这样的数据集，请查看右上角的 `API` 标签页。

这将为您提供 `数据集标识符`。

使用数据集标识符来获取特定城市 ID（`data.sfgov.org`）的特定表格——

例如，`vw6y-z8j6` 对应 [SF 311 数据](https://dev.socrata.com/foundry/data.sfgov.org/vw6y-z8j6)。

例如，`tmnf-yvry` 对应 [SF 警察数据](https://dev.socrata.com/foundry/data.sfgov.org/tmnf-yvry)。

```python
pip install -qU  sodapy
```

```python
from langchain_community.document_loaders import OpenCityDataLoader
```

```python
dataset = "vw6y-z8j6"  # 311 数据
dataset = "tmnf-yvry"  # 犯罪数据
loader = OpenCityDataLoader(city_id="data.sfgov.org", dataset_id=dataset, limit=2000)
```

```python
docs = loader.load()
```
