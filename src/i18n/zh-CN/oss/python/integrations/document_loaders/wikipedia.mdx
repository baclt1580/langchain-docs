>[维基百科](https://wikipedia.org/) 是一个多语言的免费在线百科全书，由一群被称为维基人的志愿者通过开放协作和维护，使用基于维基的编辑系统 MediaWiki。`维基百科` 是历史上规模最大、阅读量最高的参考工具。

本笔记本展示了如何将来自 `wikipedia.org` 的维基页面加载为我们下游使用的文档格式。

## 安装

首先，你需要安装 `langchain_community` 和 `wikipedia` 包。

```python
pip install -qU langchain_community wikipedia
```

## 参数

`WikipediaLoader` 具有以下参数：

- `query`：用于在维基百科中查找文档的自由文本
- `lang`（可选）：默认="en"。用于在维基百科的特定语言部分进行搜索
- `load_max_docs`（可选）：默认=100。用于限制下载的文档数量。下载全部 100 个文档需要时间，因此实验时请使用较小的数字。目前有 300 个文档的硬性限制。
- `load_all_available_meta`（可选）：默认=False。默认情况下，仅下载最重要的字段：`title` 和 `summary`。如果为 `True`，则将下载所有可用字段。
- `doc_content_chars_max`（可选）：默认=4000。文档内容的最大字符数。

## 示例

```python
from langchain_community.document_loaders import WikipediaLoader
```

```python
docs = WikipediaLoader(query="HUNTER X HUNTER", load_max_docs=2).load()
len(docs)
```
