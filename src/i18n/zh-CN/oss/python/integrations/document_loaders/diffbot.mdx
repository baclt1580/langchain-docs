>[Diffbot](https://docs.diffbot.com/docs/getting-started-with-diffbot) 是一套基于机器学习的产品，能够轻松结构化网络数据。

>Diffbot 的 [Extract API](https://docs.diffbot.com/reference/extract-introduction) 是一项服务，用于结构化和规范化网页数据。

>与传统的网络爬虫工具不同，`Diffbot Extract` 不需要任何规则来读取页面内容。它使用计算机视觉模型将页面分类为 20 种可能类型之一，然后将原始 HTML 标记转换为 JSON。生成的结构化 JSON 遵循一致的 [基于类型的本体论](https://docs.diffbot.com/docs/ontology)，这使得从多个不同网络来源以相同模式提取数据变得容易。

## 概述

本指南介绍如何使用 [Diffbot Extract API](https://www.diffbot.com/products/extract/) 从一系列 URL 中提取数据，并将其转换为可用于下游处理的结构化 JSON。

## 设置

首先安装所需的包。

```python
pip install -qU langchain-community
```

Diffbot 的 Extract API 需要一个 API 令牌。请按照以下说明 [获取免费 API 令牌](/oss/integrations/providers/diffbot#installation-and-setup)，然后设置环境变量。

```python
%env DIFFBOT_API_TOKEN REPLACE_WITH_YOUR_TOKEN
```

## 使用文档加载器

导入 DiffbotLoader 模块，并使用 URL 列表和您的 Diffbot 令牌实例化它。

```python
import os

from langchain_community.document_loaders import DiffbotLoader

urls = [
    "https://python.langchain.com/",
]

loader = DiffbotLoader(urls=urls, api_token=os.environ.get("DIFFBOT_API_TOKEN"))
```

通过 `.load()` 方法，您可以查看加载的文档

```python
loader.load()
```

## 将提取的文本转换为图文档

结构化的页面内容可以通过 `DiffbotGraphTransformer` 进一步处理，以提取实体和关系并构建成图。

```python
pip install -qU langchain-experimental
```

```python
from langchain_experimental.graph_transformers.diffbot import DiffbotGraphTransformer

diffbot_nlp = DiffbotGraphTransformer(
    diffbot_api_key=os.environ.get("DIFFBOT_API_TOKEN")
)
graph_documents = diffbot_nlp.convert_to_graph_documents(loader.load())
```

要继续将数据加载到知识图谱中，请遵循 [`DiffbotGraphTransformer` 指南](/oss/integrations/graphs/diffbot/#loading-the-data-into-a-knowledge-graph)。
