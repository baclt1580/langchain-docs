Browserless 是一项服务，允许您在云端运行无头 Chrome 实例。这是一种无需自行管理基础设施即可大规模运行基于浏览器的自动化的绝佳方式。

要将 Browserless 用作文档加载器，请按此笔记本所示初始化 `BrowserlessLoader` 实例。请注意，默认情况下，`BrowserlessLoader` 会返回页面 `body` 元素的 `innerText`。若要禁用此功能并获取原始 HTML，请将 `text_content` 设置为 `False`。

```python
from langchain_community.document_loaders import BrowserlessLoader
```

```python
BROWSERLESS_API_TOKEN = "YOUR_BROWSERLESS_API_TOKEN"
```

```python
loader = BrowserlessLoader(
    api_token=BROWSERLESS_API_TOKEN,
    urls=[
        "https://en.wikipedia.org/wiki/Document_classification",
    ],
    text_content=True,
)

documents = loader.load()

print(documents[0].page_content[:1000])
```
