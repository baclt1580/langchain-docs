> [Visio 文件](https://fr.wikipedia.org/wiki/Microsoft_Visio)（扩展名为 .vsdx）与 Microsoft Visio 相关联，后者是一款图表创建软件。它存储有关图表结构、布局和图形元素的信息。这种格式便于在商业、工程和计算机科学等领域创建和共享可视化内容。

一个 Visio 文件可以包含多个页面。其中一些页面可能作为其他页面的背景，并且这种情况可能跨越多个图层。此**加载器**会提取每个页面及其关联页面的文本内容，从而能够提取每个页面上所有可见的文本，类似于 OCR 算法所做的。

**警告**：只有扩展名为 **.vsdx** 的 Visio 文件与此加载器兼容。扩展名为 .vsd 等的文件不兼容，因为它们无法转换为压缩的 XML。

```python
from langchain_community.document_loaders import VsdxLoader
```

```python
loader = VsdxLoader(file_path="./example_data/fake.vsdx")
documents = loader.load()
```

**显示已加载的文档**

```python
for i, doc in enumerate(documents):
    print(f"\n------ 第 {doc.metadata['page']} 页 ------")
    print(f"页面标题 : {doc.metadata['page_name']}")
    print(f"来源 : {doc.metadata['source']}")
    print("\n==> 内容 <== ")
    print(doc.page_content)
```
