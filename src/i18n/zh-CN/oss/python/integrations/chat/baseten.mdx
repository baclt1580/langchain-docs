## 设置

要访问 Baseten 模型，您需要创建一个 Baseten 账户，获取 API 密钥，并安装 `langchain-baseten` 集成包。

前往 [Baseten 网站](https://app.baseten.co) 创建账户并生成 API 密钥。完成后，设置 `BASETEN_API_KEY` 环境变量：

### 凭证

```python 设置 API 密钥 icon="key"
import getpass
import os

if "BASETEN_API_KEY" not in os.environ:
    os.environ["BASETEN_API_KEY"] = getpass.getpass("输入您的 Baseten API 密钥：")
```

要启用模型调用的自动化 <Tooltip tip="记录模型执行的每一步以进行调试和改进">追踪</Tooltip>，请设置您的 [LangSmith](/langsmith/home) API 密钥：

```python 启用追踪 icon="flask"
os.environ["LANGSMITH_API_KEY"] = getpass.getpass("输入您的 LangSmith API 密钥：")
os.environ["LANGSMITH_TRACING"] = "true"
```

### 安装

LangChain Baseten 集成位于 `langchain-baseten` 包中：

<CodeGroup>
    ```python pip
    pip install -U langchain-baseten
    ```
    ```python uv

    ```
</CodeGroup>

## 实例化

Baseten 提供两种访问聊天模型的方式：

1.  **模型 API**：用于访问最新、最受欢迎的开源模型。
2.  **专用 URL**：使用具有专用资源的特定模型部署。

两种方法都支持自动端点规范化。

```python 使用模型标识符初始化 icon="robot"
from langchain_baseten import ChatBaseten

# 选项 1：使用模型 API 和模型标识符
model = ChatBaseten(
    model="moonshotai/Kimi-K2-Instruct-0905",  # 从可用的模型标识符中选择：https://docs.baseten.co/development/model-apis/overview#supported-models
    api_key="your-api-key",  # 或设置 BASETEN_API_KEY 环境变量
)
```

```python 使用模型 URL 初始化 icon="link"
from langchain_baseten import ChatBaseten

# 选项 2：使用专用部署和模型 URL
model = ChatBaseten(
    model_url="https://model-<id>.api.baseten.co/environments/production/predict",
    api_key="your-api-key",  # 或设置 BASETEN_API_KEY 环境变量
)
```

## 调用

```python 基本调用 icon="player-play"
# 使用聊天模型
response = model.invoke("你好，最近怎么样？")
print(response.content)
```

```text
你好！我很好，谢谢关心！你呢？
```
