## 设置

要访问 Reka 模型，您需要创建一个 Reka 开发者账户，获取 API 密钥，并通过 `pip install reka-api` 安装 `langchain_community` 集成包和 reka python 包。

### 凭证

前往 [platform.reka.ai/](https://platform.reka.ai/) 注册 Reka 并生成 API 密钥。完成后，设置 REKA_API_KEY 环境变量：

### 安装

LangChain ChatReka 集成位于 `langchain_community` 包中：

```python
pip install -qU langchain_community reka-api
```

# 图像输入

```python
from langchain.messages import HumanMessage

image_url = "https://v0.docs.reka.ai/_images/000000245576.jpg"

message = HumanMessage(
    content=[
        {"type": "text", "text": "描述这张图片中的天气"},
        {
            "type": "image_url",
            "image_url": {"url": image_url},
        },
    ],
)
response = model.invoke([message])
print(response.content)
```

```text
 图片显示了一个室内环境，没有可见的窗户或自然光，也没有天气状况的指示。焦点是一只坐在电脑键盘上的猫，背景包括电脑显示器和各种办公用品。
```

# 多张图像作为输入

```python
message = HumanMessage(
    content=[
        {"type": "text", "text": "这两张图片有什么区别？"},
        {
            "type": "image_url",
            "image_url": {
                "url": "https://cdn.pixabay.com/photo/2019/07/23/13/51/shepherd-dog-4357790_1280.jpg"
            },
        },
        {
            "type": "image_url",
            "image_url": {
                "url": "https://cdn.pixabay.com/photo/2024/02/17/00/18/cat-8578562_1280.jpg"
            },
        },
    ],
)
response = model.invoke([message])
print(response.content)
```

```text
 第一张图片展示了两只德国牧羊犬，一只成年犬和一只幼犬，处于一个充满活力、郁郁葱葱的绿色环境中。成年犬嘴里叼着一根大树枝，奔跑在一片看似草地的区域，幼犬紧随其后。两只狗都展现出该品种典型的显著身体特征，如尖耳朵和浓密的毛发。

第二张图片展示了一只具有引人注目的蓝眼睛的猫的特写，可能是西伯利亚猫或缅因猫等品种，处于自然的户外环境中。猫的毛色较浅，可能是白色和灰色的混合，表情比狗更内敛。背景模糊，表明焦点在猫的脸上。

总的来说，区别在于主体（两只狗 vs. 一只猫）、环境（郁郁葱葱、充满活力的草地 vs. 更柔和的户外背景）以及所描绘的整体氛围和活动（活泼好动 vs. 宁静专注）。
```

## 链式调用

```python
from langchain_core.prompts import ChatPromptTemplate

prompt = ChatPromptTemplate(
    [
        (
            "system",
            "您是一个将 {input_language} 翻译成 {output_language} 的助手。",
        ),
        ("human", "{input}"),
    ]
)

chain = prompt | model
chain.invoke(
    {
        "input_language": "英语",
        "output_language": "德语",
        "input": "我喜欢编程。",
    }
)
```

```text
AIMessage(content=' Ich liebe Programmieren.\n\n', additional_kwargs={}, response_metadata={}, id='run-ffc4ace1-b73a-4fb3-ad0f-57e60a0f9b8d-0')
```

```python
import getpass
import os

os.environ["TAVILY_API_KEY"] = getpass.getpass("输入您的 Tavily API 密钥：")
```

```python
from langchain_community.tools.tavily_search import TavilySearchResults

search = TavilySearchResults(max_results=2)
search_results = search.invoke("旧金山的天气如何")
print(search_results)
# 如果需要，我们可以创建其他工具。
# 一旦我们拥有了所有想要的工具，就可以将它们放入一个列表中，稍后引用。
tools = [search]
```

model_with_tools = model.bind_tools(tools)

# 创建智能体

现在我们已经定义了工具和 LLM，可以创建智能体了。我们将使用 LangGraph 来构建智能体。目前，我们使用高级接口来构建智能体，但 LangGraph 的好处在于，如果您想修改智能体逻辑，这个高级接口背后有一个低级别、高度可控的 API。

现在，我们可以用 LLM 和工具初始化智能体。

注意，我们传入的是 model，而不是 model_with_tools。这是因为 @[`create_agent`] 会在底层为我们调用 `.bind_tools`。

```python
from langchain.agents import create_agent


agent_executor = create_agent(model, tools)
```

现在让我们在一个应该调用工具的示例上试试

response = agent_executor.invoke({"messages": [HumanMessage(content="你好！")]})
