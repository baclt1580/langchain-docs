## 设置

langchain-gradient 使用 DigitalOcean Gradient 平台。

在 DigitalOcean 上创建一个账户，从 Gradient 平台获取 `DIGITALOCEAN_INFERENCE_KEY` API 密钥，并安装 `langchain-gradient` 集成包。

### 凭证

前往 [DigitalOcean 登录](https://cloud.digitalocean.com/login)

1.  注册/登录 DigitalOcean 云控制台
2.  进入 Gradient 平台并导航到 Serverless Inference。
3.  点击 Create model access key，输入名称，然后创建密钥。

完成此操作后，设置 `DIGITALOCEAN_INFERENCE_KEY` 环境变量：

```python
import getpass
import os

if not os.getenv("DIGITALOCEAN_INFERENCE_KEY"):
    os.environ["DIGITALOCEAN_INFERENCE_KEY"] = getpass.getpass(
        "输入您的 DIGITALOCEAN_INFERENCE_KEY API 密钥: "
    )
```

如果您希望自动追踪模型调用，也可以通过取消注释以下代码来设置您的 [LangSmith](/langsmith/home) API 密钥：

```python
os.environ["LANGSMITH_TRACING"] = "true"
os.environ["LANGSMITH_API_KEY"] = getpass.getpass("输入您的 LangSmith API 密钥: ")
```

### 安装

DigitalOcean Gradient 集成位于 `langchain-gradient` 包中：

```python
pip install -qU langchain-gradient
```

## 实例化

现在我们可以实例化我们的模型对象并生成聊天补全：

```python
from langchain_gradient import ChatGradient

llm = ChatGradient(
    model="llama3.3-70b-instruct",
    # 其他参数...
)
```

## 链式调用

我们可以像这样将我们的模型与提示模板链接起来：

```python
from langchain_core.prompts import ChatPromptTemplate

prompt = ChatPromptTemplate(
    [
        (
            "system",
            '你是一位知识渊博的助手。请仔细阅读提供的上下文并回答用户的问题。如果答案存在于上下文中，请引用相关句子。如果不存在，请回复"未在上下文中找到。"',
        ),
        ("human", "上下文: {context}\n问题: {question}"),
    ]
)

chain = prompt | llm
chain.invoke(
    {
        "context": (
            "埃菲尔铁塔位于巴黎，于1889年建成。"
            "它由古斯塔夫·埃菲尔的工程公司设计。"
            "这座塔是世界上最知名的建筑之一。"
            "自由女神像是法国赠送给美国的礼物。"
        ),
        "question": "谁设计了埃菲尔铁塔，它是什么时候建成的？",
    }
)
```
