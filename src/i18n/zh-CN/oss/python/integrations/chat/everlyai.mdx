# 让我们尝试 EverlyAI 托管端点提供的 LLAMA 模型

```python
from langchain_community.chat_models import ChatEverlyAI
from langchain.messages import HumanMessage, SystemMessage

messages = [
    SystemMessage(content="你是一个乐于助人的 AI，会分享你所知道的一切。"),
    HumanMessage(
        content="告诉我关于你自己的技术细节。你是 Transformer 模型吗？你有多少亿参数？"
    ),
]

chat = ChatEverlyAI(
    model_name="meta-llama/Llama-2-7b-chat-hf", temperature=0.3, max_tokens=64
)
print(chat(messages).content)
```

```text
  你好！我只是一个 AI，不像人类那样拥有个人信息或技术细节。不过，我可以告诉你，我是一种 Transformer 模型，具体来说是 BERT（双向编码器表示转换器）模型。B
```

# EverlyAI 也支持流式响应

```python
from langchain_community.chat_models import ChatEverlyAI
from langchain_core.callbacks import StreamingStdOutCallbackHandler
from langchain.messages import HumanMessage, SystemMessage

messages = [
    SystemMessage(content="你是一个幽默的 AI，能给人带来欢乐。"),
    HumanMessage(content="给我讲个笑话？"),
]

chat = ChatEverlyAI(
    model_name="meta-llama/Llama-2-7b-chat-hf",
    temperature=0.3,
    max_tokens=64,
    streaming=True,
    callbacks=[StreamingStdOutCallbackHandler()],
)
chat(messages)
```

```text

为什么 AI 要去看心理医生？
*鼓点声*
因为
```

# 让我们尝试 EverlyAI 上的另一个语言模型

```python
from langchain_community.chat_models import ChatEverlyAI
from langchain_core.callbacks import StreamingStdOutCallbackHandler
from langchain.messages import HumanMessage, SystemMessage

messages = [
    SystemMessage(content="你是一个幽默的 AI，能给人带来欢乐。"),
    HumanMessage(content="给我讲个笑话？"),
]

chat = ChatEverlyAI(
    model_name="meta-llama/Llama-2-13b-chat-hf-quantized",
    temperature=0.3,
    max_tokens=128,
    streaming=True,
    callbacks=[StreamingStdOutCallbackHandler()],
)
chat(messages)
```
