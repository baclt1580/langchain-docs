>[乘积量化算法 (k-NN)](https://towardsdatascience.com/similarity-search-product-quantization-b2a1a6397701) 简而言之是一种有助于压缩数据库向量的量化算法，这在涉及大型数据集时有助于语义搜索。简言之，嵌入被拆分为 M 个子空间，随后进一步进行聚类。对向量进行聚类后，质心向量会被映射到子空间中每个簇内存在的向量上。

本笔记本介绍了如何使用一个检索器，其底层使用了由 [nanopq](https://github.com/matsui528/nanopq) 包实现的乘积量化。

```python
pip install -qU langchain-community langchain-openai nanopq
```

```python
from langchain_community.embeddings.spacy_embeddings import SpacyEmbeddings
from langchain_community.retrievers import NanoPQRetriever
```

## 使用文本来创建新检索器

```python
retriever = NanoPQRetriever.from_texts(
    ["Great world", "great words", "world", "planets of the world"],
    SpacyEmbeddings(model_name="en_core_web_sm"),
    clusters=2,
    subspace=2,
)
```

## 使用检索器

我们现在可以使用检索器了！

```python
retriever.invoke("earth")
```
