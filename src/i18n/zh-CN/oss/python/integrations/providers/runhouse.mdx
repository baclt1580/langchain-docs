本页介绍如何在 LangChain 中使用 [Runhouse](https://github.com/run-house/runhouse) 生态系统。
它分为三个部分：安装和设置、大语言模型以及嵌入。

## 安装和设置
- 使用 `pip install runhouse` 安装 Python SDK
- 如果您想使用按需集群，请使用 `sky check` 检查您的云凭据

## 自托管大语言模型
对于基本自托管大语言模型，您可以使用 `SelfHostedHuggingFaceLLM` 类。对于更
自定义的大语言模型，您可以使用 `SelfHostedPipeline` 父类。

```python
from langchain_community.llms import SelfHostedPipeline, SelfHostedHuggingFaceLLM
```

有关自托管大语言模型的更详细教程，请参阅 [此笔记本](/oss/integrations/llms/runhouse)

## 自托管嵌入
通过 Runhouse，有几种方法可以在 LangChain 中使用自托管嵌入。

对于来自 Hugging Face Transformers 模型的基本自托管嵌入，您可以使用
`SelfHostedEmbedding` 类。
```python
from langchain_community.llms import SelfHostedPipeline, SelfHostedHuggingFaceLLM
```

有关自托管嵌入的更详细教程，请参阅 [此笔记本](/oss/integrations/embeddings/self-hosted)
