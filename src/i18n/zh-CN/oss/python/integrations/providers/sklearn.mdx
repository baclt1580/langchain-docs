>[scikit-learn](https://scikit-learn.org/stable/) 是一个开源的机器学习算法集合，
> 包括一些 [k 近邻](https://scikit-learn.org/stable/modules/generated/sklearn.neighbors.NearestNeighbors.html) 的实现。`SKLearnVectorStore` 封装了该实现，并增加了将向量存储持久化为 json、bson（二进制 json）或 Apache Parquet 格式的可能性。

## 安装和设置

- 使用 `pip install scikit-learn` 安装 Python 包


## 向量存储

`SKLearnVectorStore` 为 scikit-learn 包中的最近邻实现提供了一个简单的包装器，允许您将其用作向量存储。

要导入此向量存储：

```python
from langchain_community.vectorstores import SKLearnVectorStore
```

有关 SKLearnVectorStore 包装器的更详细教程，请参阅 [此笔记本](/oss/integrations/vectorstores/sklearn)。
