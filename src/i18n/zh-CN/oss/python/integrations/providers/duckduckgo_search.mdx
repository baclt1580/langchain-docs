>[DuckDuckGo 搜索](https://github.com/deedy5/ddgs) 是一个使用 `DuckDuckGo.com` 搜索引擎搜索单词、文档、图片、视频、新闻、地图和文本翻译的包。它会将文件和图片下载到本地硬盘。

## 安装和设置

你需要安装一个 Python 包：

<CodeGroup>
```bash pip
pip install ddgs
```

```bash uv
uv add ddgs
```
</CodeGroup>

## 工具

查看 [使用示例](/oss/integrations/tools/ddg)。

有两个工具可用：

```python
from langchain_community.tools import DuckDuckGoSearchRun
from langchain_community.tools import DuckDuckGoSearchResults
```
