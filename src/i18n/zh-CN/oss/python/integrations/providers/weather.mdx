>[OpenWeatherMap](https://openweathermap.org/) 是一个开源的天气服务提供商。


## 安装和设置

<CodeGroup>
```bash pip
pip install pyowm
```

```bash uv
uv add pyowm
```
</CodeGroup>

我们需要配置 OpenWeatherMap API 令牌。
