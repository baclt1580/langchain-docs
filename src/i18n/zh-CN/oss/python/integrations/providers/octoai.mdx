>[OctoAI](https://docs.octoai.cloud/docs) 提供高效计算的便捷访问
> 并允许用户将选择的 AI 模型集成到应用程序中。
> The `OctoAI` compute service helps you run, tune, and scale AI applications easily.


## 安装和设置

- 安装 `openai` Python 包：
  ```bash
  pip install openai
  ```
- 在 `OctoAI` 注册并从 [您的 OctoAI 账户页面](https://octoai.cloud/settings) 获取 API Token。


## 聊天模型

查看 [使用示例](/oss/integrations/chat/octoai)。

```python
from langchain_community.chat_models import ChatOctoAI
```
## 大语言模型

查看 [使用示例](/oss/integrations/llms/octoai)。

```python
from langchain_community.llms.octoai_endpoint import OctoAIEndpoint
```
## 嵌入模型

```python
from langchain_community.embeddings.octoai_embeddings import OctoAIEmbeddings
```
