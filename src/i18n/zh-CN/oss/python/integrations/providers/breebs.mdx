>[Breebs](https://www.breebs.com/) 是一个开放的协作知识平台。
>任何人都可以创建一个 `Breeb`，这是一个基于存储在 Google Drive 文件夹中的 PDF 的知识胶囊。
>`Breeb` 可被任何 LLM/聊天机器人使用，以提升其专业知识、减少幻觉并提供来源访问权限。
>在幕后，`Breebs` 实现了多个 `检索增强生成 (RAG)` 模型
>  以便在每次迭代中无缝提供有用的上下文。

## 检索器

```python
from langchain_classic.retrievers import BreebsRetriever
```
