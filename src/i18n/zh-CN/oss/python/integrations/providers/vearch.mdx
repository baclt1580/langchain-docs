[Vearch](https://github.com/vearch/vearch) 是一个可扩展的分布式系统，用于深度学习向量的高效相似性搜索。

# 安装和设置

Vearch Python SDK 使 vearch 可在本地使用。Vearch python sdk 可通过 pip install vearch 轻松安装。

# 向量存储

Vearch 也可用作向量存储。更多细节见 [此笔记本](/oss/integrations/vectorstores/vearch)

```python
from langchain_community.vectorstores import Vearch
```
