> [Doctran](https://github.com/psychic-api/doctran) 是一个 Python 包。它使用 LLMs 和开源
> NLP 库将原始文本转换为干净、结构化、信息密集的文档，这些文档针对向量空间检索进行了优化。你可以将 `Doctran` 视为一个黑盒，其中杂乱的字符串进去，整洁、干净的带标签字符串出来。


## 安装与设置

<CodeGroup>
```bash pip
pip install doctran
```

```bash uv
uv add doctran
```
</CodeGroup>
