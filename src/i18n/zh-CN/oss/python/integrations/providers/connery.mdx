>[Connery SDK](https://github.com/connery-io/connery-sdk) 是一个 NPM 包，它包含 SDK 和 CLI，专为插件和操作的开发而设计。
>
>CLI 自动化开发过程中的许多事情。SDK 提供 JavaScript API 用于定义插件和操作，并将它们打包成插件服务器，该服务器具有从元数据生成的标准化 REST API。
>插件服务器处理授权、输入验证和日志记录。
>您可以专注于您操作的逻辑。
>
>请参阅 [Connery SDK 文档](https://sdk.connery.io/docs/use-cases/) 中的用例和示例
