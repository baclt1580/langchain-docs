>[Robocorp](https://robocorp.com/) 帮助构建和运行在任何地方、任何规模下都能无缝运行的 Python 工作器


## 安装和设置

您需要安装 `langchain-robocorp` Python 包：

<CodeGroup>
```bash pip
pip install langchain-robocorp
```

```bash uv
uv add langchain-robocorp
```
</CodeGroup>

您的智能体应用程序需要运行中的 `Action Server` 实例才能进行通信。
请参阅 [Robocorp 快速入门](https://github.com/robocorp/robocorp#quickstart) 以了解如何设置 Action Server 并创建您的 Actions。

您可以使用 Action Server 的 `new` 命令初始化新项目。

```bash
action-server new
cd ./your-project-name
action-server start
```
