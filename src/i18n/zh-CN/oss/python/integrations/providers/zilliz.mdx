>[Zilliz Cloud](https://zilliz.com/doc/quick_start) 是专为 LF AI Milvus® 提供的完全托管云服务，

## 安装和设置

安装 Python SDK：
<CodeGroup>
```bash pip
pip install pymilvus
```

```bash uv
uv add pymilvus
```
</CodeGroup>

## 向量存储

围绕 Zilliz 索引的封装允许您将其用作向量存储，无论是用于语义搜索还是示例选择。

```python
from langchain_community.vectorstores import Milvus
```

有关 Milvus 封装器的更详细教程，请参阅 [此笔记本](/oss/integrations/vectorstores/zilliz)
