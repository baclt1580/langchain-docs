>[Git](https://en.wikipedia.org/wiki/Git) 是一个分布式版本控制系统，用于跟踪任何一组计算机文件中的更改，通常用于在软件开发期间协调程序员协作开发源代码的工作。

## 安装和设置

首先，你需要安装 `GitPython` Python 包。

<CodeGroup>
```bash pip
pip install GitPython
```

```bash uv
uv add GitPython
```
</CodeGroup>
