>[Infinity](https://github.com/michaelfeil/infinity) 允许创建文本嵌入。

## 文本嵌入模型

存在一个 Infinity 嵌入模型，您可以通过以下方式访问
```python
from langchain_community.embeddings import InfinityEmbeddings
```
有关此内容的更详细教程，请参阅 [此笔记本](/oss/integrations/embeddings/infinity)
