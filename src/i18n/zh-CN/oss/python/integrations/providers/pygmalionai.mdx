>[PygmalionAI](https://pygmalion.chat/) 是一家通过为 [Aphrodite Engine](https://github.com/PygmalionAI/aphrodite-engine) 提供推理端点来支持开源模型的公司。


## 安装和设置


<CodeGroup>
```bash pip
pip install aphrodite-engine
```

```bash uv
uv add aphrodite-engine
```
</CodeGroup>

## 大语言模型

查看 [使用示例](/oss/integrations/llms/aphrodite)。

from langchain_community.llms import Aphrodite
