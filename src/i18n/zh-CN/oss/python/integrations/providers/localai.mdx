>[LocalAI](https://localai.io/) 是免费的、开源的 OpenAI 替代品。
> `LocalAI` 充当兼容 OpenAI API 规范的本地推理即用型替换 REST API。它允许您使用消费级硬件在本地或私有部署中运行 LLMs、生成图像、
> 音频（不仅限于此），支持多种模型家族和架构。

<Info>
`langchain-localai` 是 LocalAI 的第三方集成包。它为在 LangChain 中使用 LocalAI 服务提供了一种简单的方法。
源代码可在 [GitHub](https://github.com/mkhludnev/langchain-localai) 上获取
</Info>

## 安装和设置

<CodeGroup>
```bash pip
pip install langchain-localai
```

```bash uv
uv add langchain-localai
```
</CodeGroup>
