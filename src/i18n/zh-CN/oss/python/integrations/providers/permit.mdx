title: "集成 Permit"
description: "使用 LangChain Python 与 Permit 进行集成。"
---

[Permit.io](https://permit.io/) 提供细粒度的访问控制和策略执行。借助 LangChain，您可以集成 Permit 检查，以确保只有授权用户才能访问或检索您 LLM 应用中的数据。

## 安装和设置

<CodeGroup>
```bash pip
pip install langchain-permit
pip install permit
```

```bash uv
uv add langchain-permit
uv add permit
```
</CodeGroup>

为您的 Permit PDP 和凭据设置环境变量：

```python
export PERMIT_API_KEY="your_permit_api_key"
export PERMIT_PDP_URL="http://localhost:7766"   # or your real PDP endpoint
```

请确保您的 PDP 正在运行并已配置。有关策略设置，请参阅 [Permit 文档](https://docs.permit.io/sdk/python/quickstart-python/#2-setup-your-pdp-policy-decision-point-container)。
