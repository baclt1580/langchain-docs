>[BeautifulSoup](https://www.crummy.com/software/BeautifulSoup/) 是一个用于解析 HTML 和 XML 文档的 Python 包（包括处理格式错误的标记，即未闭合的标签，因此得名于“标签汤”）。它为解析后的页面创建解析树，可用于从 HTML 中提取数据，[3] 这对网络爬虫很有用。

## 安装和设置

<CodeGroup>
```bash pip
pip install beautifulsoup4
```

```bash uv
uv add beautifulsoup4
```
</CodeGroup>
