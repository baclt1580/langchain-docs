本页面介绍如何在 LangChain 中使用 [Epsilla](https://github.com/epsilla-cloud/vectordb)。
它分为两部分：安装和设置，然后是特定 Epsilla 包装器的参考。

## 安装和设置

- 使用 `pip/pip3 install pyepsilla` 安装 Python SDK

## 包装器

### VectorStore

存在一个围绕 Epsilla 向量数据库的包装器，允许您将其用作向量存储，
无论用于语义搜索还是示例选择。

要导入此向量存储：

```python
from langchain_community.vectorstores import Epsilla
```

有关 Epsilla 包装器的更详细逐步指南，请参阅 [此笔记本](/oss/integrations/vectorstores/epsilla)
