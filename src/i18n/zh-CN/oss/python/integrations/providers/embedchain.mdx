> [Embedchain](https://github.com/embedchain/embedchain) 是一个用于创建数据管道的 RAG 框架。
> 它加载、索引、检索并同步所有数据。
>
> 它既可作为 [开源包](https://github.com/embedchain/embedchain) 提供，也可作为 [托管平台解决方案](https://app.embedchain.ai/) 提供。


## 安装和设置

使用 pip 安装包：

<CodeGroup>
```bash pip
pip install embedchain
```

```bash uv
uv add embedchain
```
</CodeGroup>
