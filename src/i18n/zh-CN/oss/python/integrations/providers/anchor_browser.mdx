[Anchor](https://anchorbrowser.io?utm=langchain) 是用于 AI 代理浏览器自动化的平台，解决了为缺乏 API 或 API 覆盖范围有限的 Web 应用程序自动化工作流的挑战。它简化了基于浏览器的自动化的创建、部署和管理，将复杂的 Web 交互转化为简单的 API 端点。

`langchain-anchorbrowser` 提供 3 个主要工具：
- `AnchorContentTool` - 用于以 Markdown 或 HTML 格式提取网页内容。
- `AnchorScreenshotTool` - 用于网页截图。
- `AnchorWebTaskTools` - 用于执行网页任务。

## 快速开始

### 安装

安装包：

<CodeGroup>
```bash pip
pip install langchain-anchorbrowser
```

```bash uv
uv add langchain-anchorbrowser
```
</CodeGroup>

### 用法

导入并使用您选定的工具。完整的 Anchor Browser 可用工具列表，请参见 [Anchor Browser 工具页面](/oss/integrations/tools/anchor_browser) 中的 **工具功能** 表。

```python
from langchain_anchorbrowser import AnchorContentTool

# Get Markdown Content for https://www.anchorbrowser.io
AnchorContentTool().invoke(
    {"url": "https://www.anchorbrowser.io", "format": "markdown"}
)
```
