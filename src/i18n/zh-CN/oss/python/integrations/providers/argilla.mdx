>[Argilla](https://argilla.io/) 是一个用于大语言模型的开源数据策展平台。
> 使用 `Argilla`，每个人都可以利用更快的数据策展
> 结合人类和机器反馈构建稳健的语言模型。`Argilla` 支持 MLOps 周期中的每一步，
> 从数据标注到模型监控。

## 安装和设置

获取您的 [API 密钥](https://platform.openai.com/account/api-keys)。

安装 Python 包：

<CodeGroup>
```bash pip
pip install argilla
```

```bash uv
uv add argilla
```
</CodeGroup>

## 回调


```python
from langchain.callbacks import ArgillaCallbackHandler
```

查看 [示例](/oss/integrations/callbacks/argilla)。
