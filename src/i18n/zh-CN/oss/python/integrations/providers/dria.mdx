> [Dria](https://dria.co/) 是公共 RAG 模型的枢纽，供开发者
> 贡献并利用共享的嵌入湖。

有关 LangChain 与 Dria 集成的更多详细信息，请查看 [此页面](https://dria.co/docs/integrations/langchain)。

## 安装和设置

您需要安装一个 Python 包：

<CodeGroup>
```bash pip
pip install dria
```

```bash uv
uv add dria
```
</CodeGroup>

您需要从 Dria 获取 API 密钥。您可以通过在 [Dria](https://dria.co/) 注册来获取它。
