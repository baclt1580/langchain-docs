> [Joplin](https://joplinapp.org/) 是一款开源笔记应用。它记录您的想法
> 并让您可以从任何设备安全地访问它们。


## 安装和设置

`` `Joplin API` `` 需要访问令牌。
您可以在 [Joplin REST API 文档](https://joplinapp.org/api/references/rest_api/) 中找到安装说明。
