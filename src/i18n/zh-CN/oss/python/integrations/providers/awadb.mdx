>[AwaDB](https://github.com/awa-ai/awadb) 是一个用于 LLM 应用中的嵌入向量搜索和存储的 AI 原生数据库。

## 安装和设置

```bash
pip install awadb
```

## 向量存储

```python
from langchain_community.vectorstores import AwaDB
```

查看 [使用示例](/oss/integrations/vectorstores/awadb)。

## 嵌入模型

```python
from langchain_community.embeddings import AwaEmbeddings
```

查看 [使用示例](/oss/integrations/embeddings/awadb)。
