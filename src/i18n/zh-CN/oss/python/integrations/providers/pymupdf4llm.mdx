---
title: "Pymupdf4llm 集成"
description: "使用 LangChain Python 与 Pymupdf4llm 进行集成。"
---

[PyMuPDF4LLM](https://pymupdf.readthedocs.io/en/latest/pymupdf4llm) 旨在让提取 PDF 内容变得更加容易，并以 LLM 和 RAG 应用所需的 Markdown 格式呈现。

[langchain-pymupdf4llm](https://github.com/lakinduboteju/langchain-pymupdf4llm) 将 PyMuPDF4LLM 集成到 LangChain 中，作为文档加载器。

```python
pip install -qU langchain-pymupdf4llm
```

```python
from langchain_pymupdf4llm import PyMuPDF4LLMLoader, PyMuPDF4LLMParser
```
