>[Cerebrium](https://docs.cerebrium.ai/cerebrium/getting-started/introduction) 是一个无服务器 GPU 基础设施提供商。
> 它提供对多个 LLM 模型的 API 访问。

请查看 [CerebriumAI 文档](https://docs.cerebrium.ai/examples/langchain) 中的示例。

## 安装和设置

- 安装一个 Python 包：
<CodeGroup>
```bash pip
pip install cerebrium
```

```bash uv
uv add cerebrium
```
</CodeGroup>

- [获取 CerebriumAI API 密钥](https://docs.cerebrium.ai/cerebrium/getting-started/installation) 并将其设置为
  环境变量（`CEREBRIUMAI_API_KEY`）


## LLMs

请查看 [使用示例](/oss/integrations/llms/cerebriumai)。


```python
from langchain_community.llms import CerebriumAI
```
