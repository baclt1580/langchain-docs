GROBID 是一个用于提取、解析和重组原始文档的机器学习库。

它设计并预期用于解析学术论文，在这方面表现尤为出色。

*注意*：如果提供给 Grobid 的文章是大型文档（例如论文），且元素数量超过一定限制，可能无法处理。

本页介绍如何使用 Grobid 为 LangChain 解析文章。

## 安装
Grobid 的安装详情描述在 https://grobid.readthedocs.io/en/latest/Install-Grobid/。
然而，通过 Docker 容器运行 Grobid 可能更简单、更少麻烦，
如 [Grobid Docker 指南](https://grobid.readthedocs.io/en/latest/Grobid-docker/) 中所述。

## 使用 Grobid 与 LangChain

一旦 Grobid 安装并运行起来（你可以通过访问 http://localhost:8070 来检查），
你就准备好了。

你现在可以使用 GrobidParser 来生成文档
```python
from langchain_community.document_loaders.parsers import GrobidParser
from langchain_community.document_loaders.generic import GenericLoader

#Produce chunks from article paragraphs
loader = GenericLoader.from_filesystem(
    "/Users/31treehaus/Desktop/Papers/",
    glob="*",
    suffixes=[".pdf"],
    parser= GrobidParser(segment_sentences=False)
)
docs = loader.load()

#Produce chunks from article sentences
loader = GenericLoader.from_filesystem(
    "/Users/31treehaus/Desktop/Papers/",
    glob="*",
    suffixes=[".pdf"],
    parser= GrobidParser(segment_sentences=True)
)
docs = loader.load()
```
块元数据将包含边界框。尽管这些解析起来有点棘手，
它们已在 https://grobid.readthedocs.io/en/latest/Coordinates-in-PDF/ 中解释
