>[DeepInfra](https://deepinfra.com/docs) 让我们能够轻松运行
> [最新的机器学习模型](https://deepinfra.com/models)。
> DeepInfra 负责处理与运行、扩展和监控
> 模型相关的所有繁重工作。用户可以专注于您的应用程序，并通过简单的 REST API 调用集成这些模型。

>DeepInfra 提供了与 LangChain 集成的 [示例](https://deepinfra.com/docs/advanced/langchain)。

本页面介绍如何在 `DeepInfra` 生态系统中在 `LangChain` 内使用。
它分为两部分：安装和设置，然后是特定 DeepInfra 包装器的参考。

## 安装和设置

- 从 [DeepInfra 网站](https://deepinfra.com/) 获取您的 DeepInfra API 密钥。
- 获取一个 DeepInfra API 密钥并将其设置为环境变量 (`DEEPINFRA_API_TOKEN`)

## 可用模型

DeepInfra 提供了一系列准备就绪的开源 LLM。

您可以查看支持
[text-generation](https://deepinfra.com/models?type=text-generation) 和
[embeddings](https://deepinfra.com/models?type=embeddings) 的模型。

您可以查看 [请求和响应参数列表](https://deepinfra.com/meta-llama/Llama-2-70b-chat-hf/api)。

聊天模型 [遵循 openai api](https://deepinfra.com/meta-llama/Llama-2-70b-chat-hf/api?example=openai-http)


## LLM

查看 [使用示例](/oss/integrations/llms/deepinfra)。

```python
from langchain_community.llms import DeepInfra
```

## 嵌入

查看 [使用示例](/oss/integrations/embeddings/deepinfra)。

```python
from langchain_community.embeddings import DeepInfraEmbeddings
```

## 聊天模型

查看 [使用示例](/oss/integrations/chat/deepinfra)。

```python
from langchain_community.chat_models import ChatDeepInfra
```
