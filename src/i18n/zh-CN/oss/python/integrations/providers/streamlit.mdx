>[Streamlit](https://streamlit.io/) 是构建和共享数据应用的更快方式。
>`Streamlit` 能在几分钟内将数据脚本转换为可共享的 Web 应用。全部使用纯 Python。无需前端经验。
>在 [streamlit.io/generative-ai](https://streamlit.io/generative-ai) 查看更多示例。

## 安装和设置

我们需要安装 the  `streamlit` Python 包：

<CodeGroup>
```bash pip
pip install streamlit
```

```bash uv
uv add streamlit
```
</CodeGroup>


## 回调

查看 [使用示例](/oss/integrations/callbacks/streamlit)。

```python
from langchain_community.callbacks import StreamlitCallbackHandler
```
