本页介绍如何在 LangChain 中使用 [vlite](https://github.com/sdan/vlite)。vlite 是一个简单快速的向量数据库，用于存储和检索嵌入。

## 安装和设置

要安装 vlite，请运行以下命令：

<CodeGroup>
```bash pip
pip install vlite
```

```bash uv
uv add vlite
```
</CodeGroup>

如需支持 PDF OCR，请安装 `vlite[ocr]` 扩展：

<CodeGroup>
```bash pip
pip install "vlite[ocr]"
```

```bash uv
uv add "vlite[ocr]"
```
</CodeGroup>

## VectorStore

vlite 提供了其向量数据库的封装，允许您将其用作语义搜索和示例选择的向量存储。

要导入 vlite 向量存储：

```python
from langchain_community.vectorstores import vlite
```
