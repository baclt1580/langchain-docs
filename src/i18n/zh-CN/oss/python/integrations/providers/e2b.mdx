>[E2B](https://e2b.dev/) 为 AI 生成的代码执行提供开源安全沙箱
> 更多信息请查看 [e2b GitHub 仓库](https://github.com/e2b-dev)。

## 安装和设置

需要安装一个 Python 包：

<CodeGroup>
```bash pip
pip install e2b_code_interpreter
```

```bash uv
uv add e2b_code_interpreter
```
</CodeGroup>
