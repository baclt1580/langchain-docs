>[Steam (维基百科)](https://en.wikipedia.org/wiki/Steam_(service)) 是由 `Valve Corporation` 开发的视频游戏数字分发服务和商店。它自动为 Valve 的游戏提供更新，并扩展到分发第三方标题。`Steam` 提供各种功能，如带有 Valve 反作弊措施的服务器匹配、社交网络和游戏流媒体服务。

>[Steam](https://store.steampowered.com/about/) 是游玩、讨论和创作游戏的终极目的地。

Steam 工具包有两个工具：

- `Game Details`
- `Recommended Games`

本指南提供了使用 Steam API 与 LangChain 的逐步教程，以根据您当前的 Steam 游戏库存检索 Steam 游戏推荐，或收集您提供的某些 Steam 游戏的信息。

## 设置

我们需要安装两个 Python 库。

## 导入

```python
pip install -qU python-steam-api python-decouple steamspypi
```

## 分配环境变量

要使用此工具包，请准备好您的 OpenAI API 密钥、Steam API 密钥（来自 [Steam 开发者 API 密钥页面](https://steamcommunity.com/dev/apikey)）并保存好您自己的 SteamID。一旦收到 Steam API 密钥，您可以将其作为环境变量输入如下。
工具包将读取 "STEAM_KEY" API 密钥作为环境变量来验证您的身份，因此请在此处设置它们。您还需要设置 "OPENAI_API_KEY" 和 "STEAM_ID"。

```python
import os

os.environ["STEAM_KEY"] = ""
os.environ["STEAM_ID"] = ""
os.environ["OPENAI_API_KEY"] = ""
```

## 初始化

初始化 LLM、SteamWebAPIWrapper、SteamToolkit 以及最重要的 langchain agent 来处理您的查询！

## 示例

```python
from langchain_community.agent_toolkits.steam.toolkit import SteamToolkit
from langchain_community.utilities.steam import SteamWebAPIWrapper

steam = SteamWebAPIWrapper()
tools = [steam.run]
```

```python
from langchain.agents import create_agent

agent = create_agent("gpt-4.1-mini", tools)
```

```python
events = agent.stream(
    {"messages": [("user", "can you give the information about the game Terraria?")]},
    stream_mode="values",
)
for event in events:
    event["messages"][-1].pretty_print()
```
