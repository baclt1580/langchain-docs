>[GraphQL](https://graphql.org/) 是一种用于 API 的查询语言，也是针对您的数据执行这些查询的运行时。`GraphQL` 提供了您 API 中数据的完整且易于理解的描述，赋予客户端仅请求所需内容的能力，使 API 随时间演进变得更加容易，并支持强大的开发者工具。

通过将 `BaseGraphQLTool` 包含在提供给 Agent 的工具列表中，您可以授予您的 Agent 从 GraphQL API 查询数据以用于任何您需要的目的的能力。

本 Jupyter Notebook 演示了如何在 Agent 中使用 `GraphQLAPIWrapper` 组件。

在本示例中，我们将使用位于以下端点的公共 `Star Wars GraphQL API`：[swapi-graphql.netlify.app/graphql](https://swapi-graphql.netlify.app/graphql) 。

首先，您需要安装 `httpx` 和 `gql` Python 包。

```python
pip install httpx gql > /dev/null
```

```python
pip install -qU  langchain-community
```

现在，让我们使用指定的 Star Wars API 端点创建一个 BaseGraphQLTool 实例，并使用该工具初始化一个 Agent。

```python
import os

os.environ["OPENAI_API_KEY"] = ""
```

```python
from langchain_community.agent_toolkits.load_tools import load_tools

tools = load_tools(
    ["graphql"],
    graphql_endpoint="https://swapi-graphql.netlify.app/graphql",
)
```

```python
from langchain.agents import create_agent


agent = create_agent("gpt-4.1-mini", tools)
```

现在，我们可以使用 Agent 对 Star Wars GraphQL API 运行查询。让我们要求 Agent 列出所有 Star Wars 电影及其发布日期。

```python
graphql_fields = """allFilms {
    films {
      title
      director
      releaseDate
      speciesConnection {
        species {
          name
          classification
          homeworld {
            name
          }
        }
      }
    }
  }

"""

suffix = "Search for the titles of all the stawars films stored in the graphql database that has this schema "

input_message = {
    "role": "user",
    "content": suffix + graphql_fields,
}

for step in agent.stream(
    {"messages": [input_message]},
    stream_mode="values",
):
    step["messages"][-1].pretty_print()
```
