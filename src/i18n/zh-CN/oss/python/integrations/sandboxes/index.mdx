沙箱提供隔离的执行环境，用于安全地运行智能体生成的代码。了解更多关于 [沙箱](/oss/deepagents/sandboxes) 的信息。

| 提供商 | 包 |
|----------|---------|
| [`AgentCoreSandbox`](/oss/integrations/sandboxes/aws) | [`langchain-agentcore-codeinterpreter`](https://pypi.org/project/langchain-agentcore-codeinterpreter/) |
| [`DaytonaSandbox`](/oss/integrations/sandboxes/daytona) | [`langchain-daytona`](https://pypi.org/project/langchain-daytona/) |
| [`ModalSandbox`](/oss/integrations/sandboxes/modal) | [`langchain-modal`](https://pypi.org/project/langchain-modal/) |
| [`RunloopSandbox`](/oss/integrations/sandboxes/runloop) | [`langchain-runloop`](https://pypi.org/project/langchain-runloop/) |
