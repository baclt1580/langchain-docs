>[PromptLayer](https://docs.promptlayer.com/introduction) 是一个用于提示工程的平台。它还能通过可视化请求、版本化提示和追踪使用情况来增强 LLM 的可观测性。
>
>虽然 `PromptLayer` 确实提供了直接与 LangChain 集成的 LLM（例如 [`PromptLayerOpenAI`](/oss/integrations/llms/promptlayer_openai)），但使用回调是与 LangChain 集成 `PromptLayer` 的推荐方式。

在本指南中，我们将介绍如何设置 `PromptLayerCallbackHandler`。

更多信息请参阅 [PromptLayer 文档](https://docs.promptlayer.com/languages/langchain)。

## 安装与设置

```python
pip install -qU  langchain-community promptlayer --upgrade
```

### 获取 API 凭证

如果您没有 PromptLayer 账户，请在 [promptlayer.com](https://www.promptlayer.com) 上创建一个。然后，通过点击导航栏中的设置齿轮图标获取 API 密钥，并将其设置为名为 `PROMPTLAYER_API_KEY` 的环境变量。

## 使用方法

开始使用 `PromptLayerCallbackHandler` 相当简单，它接受两个可选参数：

1. `pl_tags` - 一个可选的字符串列表，将在 PromptLayer 上作为标签进行追踪。
2. `pl_id_callback` - 一个可选函数，该函数将以 `promptlayer_request_id` 作为参数。此 ID 可与 PromptLayer 的所有追踪功能一起使用，以追踪元数据、评分和提示使用情况。

## 简单的 OpenAI 示例

在这个简单示例中，我们将 `PromptLayerCallbackHandler` 与 @[`ChatOpenAI`] 一起使用。我们添加了一个名为 `chatopenai` 的 PromptLayer 标签。

```python
import promptlayer  # 别忘了这个 🍰
from langchain_community.callbacks.promptlayer_callback import (
    PromptLayerCallbackHandler,
)
```

```python
from langchain.messages import HumanMessage
from langchain_openai import ChatOpenAI

chat_llm = ChatOpenAI(
    temperature=0,
    callbacks=[PromptLayerCallbackHandler(pl_tags=["chatopenai"])],
)
llm_results = chat_llm.invoke(
    [
        HumanMessage(content="1,2,3 后面是什么？"),
        HumanMessage(content="再讲一个笑话？"),
    ]
)
print(llm_results)
```

## GPT4All 示例

```python
from langchain_community.llms import GPT4All

model = GPT4All(model="./models/gpt4all-model.bin", n_ctx=512, n_threads=8)
callbacks = [PromptLayerCallbackHandler(pl_tags=["langchain", "gpt4all"])]

response = model.invoke(
    "从前，",
    config={"callbacks": callbacks},
)
```

## 完整功能示例

在这个示例中，我们将解锁 `PromptLayer` 的更多功能。

PromptLayer 允许您可视化地创建、版本化和追踪提示模板。使用 [提示注册表](https://docs.promptlayer.com/features/prompt-registry)，我们可以通过编程方式获取名为 `example` 的提示模板。

我们还定义了一个 `pl_id_callback` 函数，该函数接收 `promptlayer_request_id` 并记录评分、元数据，并关联所使用的提示模板。有关追踪的更多信息，请阅读[我们的文档](https://docs.promptlayer.com/features/prompt-history/request-id)。

```python
from langchain_openai import OpenAI


def pl_id_callback(promptlayer_request_id):
    print("prompt layer id ", promptlayer_request_id)
    promptlayer.track.score(
        request_id=promptlayer_request_id, score=100
    )  # 评分是一个 0-100 的整数
    promptlayer.track.metadata(
        request_id=promptlayer_request_id, metadata={"foo": "bar"}
    )  # 元数据是一个键值对字典，会在 PromptLayer 上被追踪
    promptlayer.track.prompt(
        request_id=promptlayer_request_id,
        prompt_name="example",
        prompt_input_variables={"product": "toasters"},
        version=1,
    )  # 将请求关联到一个提示模板


openai_llm = OpenAI(
    model_name="gpt-3.5-turbo-instruct",
    callbacks=[PromptLayerCallbackHandler(pl_id_callback=pl_id_callback)],
)

example_prompt = promptlayer.prompts.get("example", version=1, langchain=True)
openai_llm.invoke(example_prompt.format(product="toasters"))
```

这样就完成了！设置之后，您的所有请求都会显示在 PromptLayer 仪表板上。
此回调也适用于 LangChain 上实现的任何 LLM。
