## 概述

本指南将帮助您集成并使用 [Mixedbread AI](https://mixedbread.ai/) 的重排序 API。该重排序 API 允许您根据给定查询对文档列表进行重新排序，从而提高搜索结果或任何排名列表的相关性。

## 安装

首先，安装 `@langchain/mixedbread-ai` 包：

<Tip>
有关安装 LangChain 包的通用说明，请参阅[此部分](/oss/langchain/install)。
</Tip>

```bash
npm install @langchain/mixedbread-ai
```

## 认证

在 [Mixedbread AI](https://mixedbread.ai/) 注册以获取您的 API 密钥。然后，您可以将 `MXBAI_API_KEY` 环境变量设置为您的 Mixedbread AI API 密钥，或在构造类时直接将其作为 `apiKey` 选项传递。

## 使用重排序

`MixedbreadAIReranker` 类提供了对重排序 API 的访问。以下是使用方法：

1. **导入类**：首先，从包中导入 `MixedbreadAIReranker` 类。

```typescript
import { MixedbreadAIReranker } from "@langchain/mixedbread-ai";
```

2. **实例化类**：使用您的 API 密钥创建 `MixedbreadAIReranker` 的实例。

```typescript
const reranker = new MixedbreadAIReranker({ apiKey: "your-api-key" });
```

3. **重排序文档**：使用 `rerankDocuments` 方法根据查询对文档进行重新排序。

```typescript
const documents = [
  { pageContent: "烤面包需要面粉" },
  { pageContent: "烤面包需要酵母" },
  { pageContent: "吃面包只需要好胃口" },
];
const query = "烤面包需要什么？";
const result = await reranker.compressDocuments(documents, query);
console.log(result);
```
