description: "使用 LangChain JavaScript 与 Wolframalpha 工具集成。"
---

WolframAlpha 工具可将您的智能体和链连接到 WolframAlpha 先进的计算智能引擎。

## 设置

您需要在 [WolframAlpha 开发者门户](https://developer.wolframalpha.com/) 创建一个应用并获取 `appid`。

## 使用方法

```typescript
import { WolframAlphaTool } from "@langchain/community/tools/wolframalpha";

const tool = new WolframAlphaTool({
  appid: "YOUR_APP_ID",
});

const res = await tool.invoke("What is 2 * 2?");

console.log(res);
```
