[Tavily](https://tavily.com/) 是一款专为 AI 智能体（大语言模型）构建的搜索引擎，能够快速提供实时、准确且基于事实的搜索结果。Tavily 提供四个关键端点，其中之一是 Extract（提取）端点，它可以从指定 URL 提取原始内容。

本指南提供了快速入门 Tavily [工具](/oss/integrations/tools/) 的概览。关于 Tavily 工具的完整解析，您可以在 [API 参考](https://reference.langchain.com/javascript/langchain-tavily/TavilyExtract) 中找到更详细的文档。

## 设置

该集成位于 `@langchain/tavily` 包中，您可以按如下所示安装：

<CodeGroup>
```bash npm
npm install @langchain/tavily @langchain/core
```
```bash yarn
yarn add @langchain/tavily @langchain/core
```
```bash pnpm
pnpm add @langchain/tavily @langchain/core
```
</CodeGroup>

### 凭证

[设置一个 Tavily API 密钥](https://app.tavily.com) 并将其设置为名为 `TAVILY_API_KEY` 的环境变量。

```typescript
process.env.TAVILY_API_KEY = "YOUR_API_KEY"
```

同时设置 [LangSmith](https://smith.langchain.com/) 以获得最佳的观测性体验（非必需但很有帮助）：

```typescript
process.env.LANGSMITH_TRACING="true"
process.env.LANGSMITH_API_KEY="your-api-key"
```

## 实例化

您可以像这样导入并实例化 `TavilyExtract` 工具：

```typescript
import { TavilyExtract } from "@langchain/tavily";

const tool = new TavilyExtract({
  extractDepth: "basic",
  includeImages: false,
});
```

## 调用

### [直接使用参数调用](/oss/langchain/tools)

Tavily Extract 工具在调用时接受以下参数：

* `urls`（必需）：要从中提取内容的 URL 列表。

* `extractDepth` 和 `includeImages` 也可以在调用时设置。

```typescript
await tool.invoke({
  "urls": ["https://en.wikipedia.org/wiki/Lionel_Messi"]
});
```

### [使用 ToolCall 调用](/oss/langchain/tools)

我们也可以使用模型生成的 `ToolCall` 来调用该工具，此时将返回一个 @[`ToolMessage`]：

```typescript
// 这通常由模型生成，但为了演示目的，我们将直接创建一个工具调用。
const modelGeneratedToolCall = {
  args: {"urls": ["https://en.wikipedia.org/wiki/Lionel_Messi"]},
  id: "1",
  name: tool.name,
  type: "tool_call",
}

await tool.invoke(modelGeneratedToolCall)
```

## 链式调用

我们可以通过先将工具绑定到一个 [支持工具调用的模型](/oss/langchain/tools/)，然后在链中使用它：

```typescript
// @lc-docs-hide-cell

import { ChatOpenAI } from "@langchain/openai"

const llm = new ChatOpenAI({
  model: "gpt-4.1",
  temperature: 0,
})
```

```typescript
import { HumanMessage } from "@langchain/core/messages";
import { ChatPromptTemplate } from "@langchain/core/prompts";
import { RunnableLambda } from "@langchain/core/runnables";

const prompt = ChatPromptTemplate.fromMessages(
  [
    ["system", "你是一个有用的助手。"],
    ["placeholder", "{messages}"],
  ]
)

const llmWithTools = llm.bindTools([tool]);

const chain = prompt.pipe(llmWithTools);

const toolChain = RunnableLambda.from(
  async (userInput: string, config) => {
    const humanMessage = new HumanMessage(userInput);
    const aiMsg = await chain.invoke({
      messages: [new HumanMessage(userInput)],
    }, config);
    const toolMsgs = await tool.batch(aiMsg.tool_calls, config);
    return chain.invoke({
      messages: [humanMessage, aiMsg, ...toolMsgs],
    }, config);
  }
);

const toolChainResult = await toolChain.invoke("['https://en.wikipedia.org/wiki/Albert_Einstein','https://en.wikipedia.org/wiki/Theoretical_physics']");
```

```typescript
const { tool_calls, content } = toolChainResult;

console.log("AIMessage", JSON.stringify({
  tool_calls,
  content,
}, null, 2));
```
