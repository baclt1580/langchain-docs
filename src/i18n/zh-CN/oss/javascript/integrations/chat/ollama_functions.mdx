<Warning>
**LangChain Ollama 集成包已官方支持工具调用功能。[查看 Ollama 工具调用文档](/oss/integrations/chat/ollama#tools)。**

</Warning>

LangChain 提供了一个实验性封装器，用于通过 [Ollama](https://github.com/jmorganca/ollama) 本地运行的开源模型，使其具备与 OpenAI 函数相同的 API。

请注意，功能更强大、能力更强的模型在处理复杂模式或多项函数时表现会更佳。以下示例使用 [Mistral](https://ollama.ai/library/mistral) 模型。

<Warning>
**这是一个实验性封装器，尝试为原生不支持工具调用的模型添加工具调用支持。请谨慎使用。**

</Warning>

## 设置

按照 [这些说明](https://github.com/jmorganca/ollama) 设置并运行本地 Ollama 实例。

## 初始化模型

你可以像初始化标准 `ChatOllama` 实例一样初始化此封装器：

```typescript
import { OllamaFunctions } from "@langchain/community/experimental/chat_models/ollama_functions";

const model = new OllamaFunctions({
  temperature: 0.1,
  model: "mistral",
});
```

## 传入函数

现在你可以像使用 OpenAI 一样传入函数：

```typescript
import { ChatOllama } from "@langchain/ollama";
import { HumanMessage } from "@langchain/core/messages";

const model = new ChatOllama({
  temperature: 0.1,
  model: "mistral",
})
  .bindTools([
    {
      name: "get_current_weather",
      description: "获取指定地点的当前天气",
      parameters: {
        type: "object",
        properties: {
          location: {
            type: "string",
            description: "城市和州，例如：San Francisco, CA",
          },
          unit: { type: "string", enum: ["celsius", "fahrenheit"] },
        },
        required: ["location"],
      },
    },
  ])
  .withConfig({
    // 你可以设置 `tool_choice` 参数来强制模型使用特定函数
    tool_choice: "get_current_weather",
  });

const response = await model.invoke([
  new HumanMessage({
    content: "波士顿的天气怎么样？",
  }),
]);

console.log(response);

/*
  AIMessage {
    content: '',
    additional_kwargs: {
      function_call: {
        name: 'get_current_weather',
        arguments: '{"location":"Boston, MA","unit":"fahrenheit"}'
      }
    }
  }
*/
```

## 用于信息提取

```typescript
import * as z from "zod";

import { ChatOllama } from "@langchain/ollama";
import { PromptTemplate } from "@langchain/core/prompts";
import { JsonOutputFunctionsParser } from "@langchain/core/output_parsers/openai_functions";

const EXTRACTION_TEMPLATE = `提取并保存以下段落中提到的相关实体及其属性。

段落：
{input}
`;

const prompt = PromptTemplate.fromTemplate(EXTRACTION_TEMPLATE);

// 使用 Zod 简化模式声明
const schema = z.object({
  people: z.array(
    z.object({
      name: z.string().describe("人物姓名"),
      height: z.number().describe("人物身高"),
      hairColor: z.optional(z.string()).describe("人物发色"),
    })
  ),
});

const model = new ChatOllama({
  temperature: 0.1,
  model: "mistral",
})
  .bindTools([
    {
      name: "information_extraction",
      description: "从段落中提取相关信息。",
      schema,
    },
  ])
  .withConfig({
    tool_choice: "information_extraction",
  });

// 使用 JsonOutputFunctionsParser 直接获取解析后的 JSON 响应。
const chain = prompt.pipe(model).pipe(new JsonOutputFunctionsParser());

const response = await chain.invoke({
  input:
    "Alex 身高 5 英尺。Claudia 比 Alex 高 1 英尺，跳得也比他高。Claudia 有一头橙发，Alex 是金发。",
});

console.log(JSON.stringify(response, null, 2));

/*
{
  "people": [
    {
      "name": "Alex",
      "height": 5,
      "hairColor": "blonde"
    },
    {
      "name": "Claudia",
      "height": {
        "$num": 1,
        "add": [
          {
            "name": "Alex",
            "prop": "height"
          }
        ]
      },
      "hairColor": "orange"
    }
  ]
}
*/
```

## 自定义

在底层，此封装器使用 Ollama 的 JSON 模式来约束输出为 JSON 格式，然后将工具模式作为 JSON 模式传入提示中。

由于不同模型有各自的优势，传入自定义系统提示可能会有所帮助。以下是一个示例：

```typescript
import { ChatOllama } from "@langchain/ollama";
import { HumanMessage, SystemMessage } from "@langchain/core/messages";

// 自定义系统提示以格式化工具。你必须鼓励模型
// 将输出包装在具有 "tool" 和 "tool_input" 属性的 JSON 对象中。
const toolSystemPromptTemplate = `你可以使用以下工具：

{tools}

要使用工具，请使用以下结构的 JSON 对象进行响应：
{{
  "tool": <被调用工具的名称>,
  "tool_input": <符合上述 JSON 模式的工具参数>
}}`;

const model = new ChatOllama({
  temperature: 0.1,
  model: "mistral",
})
  .bindTools([
    {
      name: "get_current_weather",
      description: "获取指定地点的当前天气",
      parameters: {
        type: "object",
        properties: {
          location: {
            type: "string",
            description: "城市和州，例如：San Francisco, CA",
          },
          unit: { type: "string", enum: ["celsius", "fahrenheit"] },
        },
        required: ["location"],
      },
    },
  ])
  .withConfig({
    // 你可以设置 `tool_choice` 参数来强制模型使用特定函数
    tool_choice: "get_current_weather",
  });

const response = await model.invoke([
  new SystemMessage(toolSystemPromptTemplate),
  new HumanMessage({
    content: "波士顿的天气怎么样？",
  }),
]);

console.log(response);

/*
  AIMessage {
    content: '',
    additional_kwargs: {
      function_call: {
        name: 'get_current_weather',
        arguments: '{"location":"Boston, MA","unit":"fahrenheit"}'
      }
    }
  }
*/
```
