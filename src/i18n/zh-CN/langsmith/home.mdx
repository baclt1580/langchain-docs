**LangSmith 是一个框架无关的平台，用于构建、调试和部署 AI 智能体与 LLM 应用。** 在一个地方，与您的智能体技术栈一起，追踪请求、评估输出、测试提示词并管理部署。

## 开始使用

<Steps>
    <Step title="创建账户" icon="user-plus">
        在 [smith.langchain.com](https://smith.langchain.com) 注册（无需信用卡）。
        您可以使用 **Google**、**GitHub** 或 **邮箱** 登录。
    </Step>
    <Step title="创建 API 密钥" icon="key">
        前往您的 [设置页面](https://smith.langchain.com/settings) → **API 密钥** → **创建 API 密钥**。
        复制密钥并安全保存。
    </Step>
    <Step title="选择集成方式" icon="plug">
        LangSmith 可与多种框架和提供商协同工作。浏览 [可用集成](/langsmith/integrations) 以连接您的技术栈。
    </Step>
</Steps>

准备好您的账户和 API 密钥后，选择一个快速入门指南开始使用 LangSmith 进行构建：
