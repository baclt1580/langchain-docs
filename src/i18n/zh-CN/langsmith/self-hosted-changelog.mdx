## langsmith-0.13.37

- 修复了追踪会话，使用改进后的 limit+1 近似值。
- 在前端添加了单选卡片组件，以实现更好的 UI 交互。
- 改进了提示词提交页面上工具的显示。
- 将“保存提示词”模态框迁移到使用设计系统的对话框，以实现一致的外观。
- 修复了见解中导致结果不正确的运算符优先级错误。
- 为 LLM 身份验证代理添加了 URL 白名单支持，以增强安全性。
- 默认情况下，为使用 Go 队列的自托管部署启用了审计日志。
- 对 OAuth 令牌元数据列强制设置了 NOT NULL DEFAULT 约束。
- 将 PAQ smithdb 与 smith-go 后端集成，以增强功能。
- 在前端强制实施了 MCP 服务器 RBAC，以提高安全性。
- 在沙盒中添加了 x 服务身份验证支持，以改善集成。
- 确保 SSRF 策略允许本地开发环境的 localhost。
- 增强了反馈系统，确保所有端点都能更新。
- 修复了本地开发和身份验证中 hub 的内存检查问题。
- 为沙盒添加了类型化的代理头模型，包含不同类型以增强安全性。
- 改进了前端对反馈增量的处理。
- 当组织/项目变更时删除 MCP 服务器，以维护系统完整性。
