---
title: 如何自定义 Dockerfile
sidebarTitle: 自定义 Dockerfile
---
用户可以在从父 LangGraph 镜像导入后，向 Dockerfile 添加一系列额外的行。为此，只需修改 `langgraph.json` 文件，将需要运行的命令传入 `dockerfile_lines` 键即可。例如，如果我们希望在图中使用 `Pillow`，就需要添加以下依赖项：

```
{
    "dependencies": ["."],
    "graphs": {
        "openai_agent": "./openai_agent.py:agent",
    },
    "env": "./.env",
    "dockerfile_lines": [
        "RUN apt-get update && apt-get install -y libjpeg-dev zlib1g-dev libpng-dev",
        "RUN pip install Pillow"
    ]
}
```

这样，如果我们需要处理 `jpeg` 或 `png` 图像格式，就会安装 Pillow 所需的系统包。
