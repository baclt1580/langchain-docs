---
title: 联合营销
---

LangChain 每月下载量超过 6000 万次，拥有大量正在构建 LLM 应用程序的开发者受众。除了简单罗列集成方案，我们更致力于推广高质量、富有教育意义的示例，以启发开发者并推动生态系统发展。

<Note>
    虽然我们偶尔会分享集成方案，但我们优先考虑那些能提供深刻见解和最佳实践的内容。我们的主要社交媒体渠道是 [Twitter](https://x.com/LangChain) 和 [LinkedIn](https://www.linkedin.com/company/langchain/)，我们会在这些平台上重点展示最优秀的示例。
</Note>

### 我们乐于推广的内容类型

<AccordionGroup>
    <Accordion title="教育性内容" icon="school">
        展示教育性内容的博客、YouTube 视频及其他媒体形式。请注意，我们更倾向于内容不以“如何使用集成方案 XYZ”为框架，而是以“如何实现 ABC”为导向，因为我们发现这对开发者更具教育意义和实际帮助。
    </Accordion>

    <Accordion title="端到端应用程序" icon="cube">
        端到端应用程序是寻求构建应用的开发者的绝佳资源。我们更倾向于展示那些本质上更复杂/更具智能体特性，并使用 [LangGraph](https://github.com/langchain-ai/langgraph) 作为编排框架的应用程序。以下类型的内容尤其让我们兴奋：



        - 长期记忆系统
        - 人机交互循环模式
        - 多智能体架构
    </Accordion>

    <Accordion title="研究性内容" icon="flask">
        我们非常乐意推广新颖的研究成果！无论是基于 LangChain 构建的研究，还是与 LangChain 集成的研究。
    </Accordion>
</AccordionGroup>
