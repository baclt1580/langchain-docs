>[College Confidential](https://www.collegeconfidential.com/) 提供 3,800 多所学院和大学的信息。

本文介绍如何将 `College Confidential` 网页加载成可供下游使用的文档格式。

```python
from langchain_community.document_loaders import CollegeConfidentialLoader
```

```python
loader = CollegeConfidentialLoader(
    "https://www.collegeconfidential.com/colleges/brown-university/"
)
```

```python
data = loader.load()
```
