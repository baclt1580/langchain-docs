>[Telegram Messenger](https://web.telegram.org/a/) 是一个全球可访问的免费增值、跨平台、加密、基于云且中心化的即时通讯服务。该应用还提供可选的端到端加密聊天和视频通话、VoIP、文件共享以及其他多项功能。

本笔记本介绍了如何从 `Telegram` 加载数据，并将其转换为可被 LangChain 处理的格式。

```python
from langchain_community.document_loaders import (
    TelegramChatApiLoader,
    TelegramChatFileLoader,
)
```

```python
loader = TelegramChatFileLoader("example_data/telegram.json")
```

```python
loader.load()
```

`TelegramChatApiLoader` 直接从 Telegram 中指定的聊天加载数据。为了导出数据，您需要验证您的 Telegram 账户。

您可以从 [my.telegram.org/auth?to=apps](https://my.telegram.org/auth?to=apps) 获取 API_HASH 和 API_ID。

chat_entity——建议使用频道的 [实体](https://docs.telethon.dev/en/stable/concepts/entities.html?highlight=Entity#what-is-an-entity)。

```python
loader = TelegramChatApiLoader(
    chat_entity="<CHAT_URL>",  # 建议在此处使用 Entity
    api_hash="<API HASH >",
    api_id="<API_ID>",
    username="",  # 仅用于缓存会话。
)
```

```python
loader.load()
```
