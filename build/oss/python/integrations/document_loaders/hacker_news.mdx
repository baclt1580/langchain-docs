>[Hacker News](https://en.wikipedia.org/wiki/Hacker_News)（有时缩写为 `HN`）是一个专注于计算机科学和创业的社交新闻网站。它由投资基金和初创企业孵化器 `Y Combinator` 运营。通常，可提交的内容被定义为“任何能满足人们求知欲的事物”。

本笔记本介绍了如何从 [Hacker News](https://news.ycombinator.com/) 拉取页面数据和评论。

```python
from langchain_community.document_loaders import HNLoader
```

```python
loader = HNLoader("https://news.ycombinator.com/item?id=34817881")
```

```python
data = loader.load()
```

```python
data[0].page_content[:300]
```

```text

```
