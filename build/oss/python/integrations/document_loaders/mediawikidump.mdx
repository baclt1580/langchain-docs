>[MediaWiki XML 转储文件](https://www.mediawiki.org/wiki/Manual:Importing_XML_dumps)包含维基的内容（所有修订版本的维基页面），但不包含站点相关数据。XML 转储不会创建维基数据库的完整备份，转储文件中不包含用户账户、图片、编辑日志等。

本文介绍如何将 MediaWiki XML 转储文件加载为我们可在下游使用的文档格式。

它使用来自 `mediawiki-utilities` 的 `mwxml` 进行转储，并使用来自 `earwig` 的 `mwparserfromhell` 来解析 MediaWiki 维基代码。

转储文件可通过 dumpBackup.php 或在维基的 Special:Statistics 页面获取。

```python
# mediawiki-utilities 在未合并的分支中支持 XML 模式 0.11
pip install -qU git+https://github.com/mediawiki-utilities/python-mwtypes@updates_schema_0.11
# mediawiki-utilities 的 mwxml 存在一个错误，修复 PR 待处理
pip install -qU git+https://github.com/gdedrouas/python-mwxml@xml_format_0.11
pip install -qU mwparserfromhell
```

```python
from langchain_community.document_loaders import MWDumpLoader
```

```python
loader = MWDumpLoader(
    file_path="example_data/testmw_pages_current.xml",
    encoding="utf8",
    # namespaces = [0,2,3] 可选列表，仅加载特定命名空间。默认加载所有命名空间。
    skip_redirects=True,  # 将跳过仅重定向到其他页面的页面（若为 False 则不跳过）
    stop_on_error=False,  # 将跳过导致解析错误的页面（若为 False 则不跳过）
)
documents = loader.load()
print(f"您的数据中包含 {len(documents)} 个文档")
```
