MHTML 既用于电子邮件，也用于网页存档。MHTML，有时也称为 MHT，代表 MIME HTML，它是一个将整个网页存档的单一文件。当以 MHTML 格式保存网页时，该文件扩展名将包含 HTML 代码、图像、音频文件、Flash 动画等。

```python
from langchain_community.document_loaders import MHTMLLoader
```

```python
# 为 MHTML 文件创建一个新的加载器对象
loader = MHTMLLoader(
    file_path="../../../../../../tests/integration_tests/examples/example.mht"
)

# 从文件中加载文档
documents = loader.load()

# 打印文档以查看结果
for doc in documents:
    print(doc)
```
