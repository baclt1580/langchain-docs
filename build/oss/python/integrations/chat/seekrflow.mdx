本指南提供了使用 `ChatSeekrFlow` [聊天模型](/oss/python/langchain/models) 的快速入门概述。

## 概述

`ChatSeekrFlow` 类封装了 SeekrFlow 上托管的聊天模型端点，实现了与 LangChain 应用的无缝集成。

### 支持的方法

`ChatSeekrFlow` 支持 `ChatModel` 的所有方法，**异步 API 除外**。

### 端点要求

`ChatSeekrFlow` 封装的 serving 端点**必须**具有 OpenAI 兼容的聊天输入/输出格式。可用于：

1. **微调的 Seekr 模型**
2. **自定义 SeekrFlow 模型**
3. **使用 Seekr 检索系统的 RAG 增强模型**

异步使用请参考 `AsyncChatSeekrFlow`（即将推出）。

# 在 LangChain 中开始使用 ChatSeekrFlow

本笔记本介绍如何在 LangChain 中使用 SeekrFlow 作为聊天模型。

## 设置

确保已安装必要的依赖项：

```bash
pip install seekrai langchain langchain-community
```

您还需要一个来自 Seekr 的 API 密钥来验证请求。

```python
# 标准库
import getpass
import os

# 第三方库
from langchain.prompts import ChatPromptTemplate
from langchain.schema import HumanMessage
from langchain_core.runnables import RunnableSequence

# OSS SeekrFlow 集成
from langchain_seekrflow import ChatSeekrFlow
from seekrai import SeekrFlow
```

## API 密钥设置

您需要将 API 密钥设置为环境变量以验证请求。

运行以下单元格。

或在运行查询前手动分配：

```python
SEEKR_API_KEY = "your-api-key-here"
```

```python
os.environ["SEEKR_API_KEY"] = getpass.getpass("输入您的 Seekr API 密钥：")
```

## 实例化

```python
os.environ["SEEKR_API_KEY"]
seekr_client = SeekrFlow(api_key=SEEKR_API_KEY)

llm = ChatSeekrFlow(
    client=seekr_client, model_name="meta-llama/Meta-Llama-3-8B-Instruct"
)
```

## 调用

```python
response = llm.invoke([HumanMessage(content="Hello, Seekr!")])
print(response.content)
```

```text
Hello there! I'm Seekr, nice to meet you! What brings you here today? Do you have a question, or are you looking for some help with something? I'm all ears (or rather, all text)!
```

## 链式调用

```python
prompt = ChatPromptTemplate.from_template("翻译成法语：{text}")

chain: RunnableSequence = prompt | llm
result = chain.invoke({"text": "Good morning"})
print(result)
```

```text
content='The translation of "Good morning" in French is:\n\n"Bonne journée"' additional_kwargs={} response_metadata={}
```

## 错误处理与调试

```python
# 定义一个最小化的模拟 SeekrFlow 客户端
class MockSeekrClient:
    """模拟 SeekrFlow API 客户端，模仿真实 API 结构。"""

    class MockChat:
        """模拟 Chat 对象，包含 completions 方法。"""

        class MockCompletions:
            """模拟 Completions 对象，包含 create 方法。"""

            def create(self, *args, **kwargs):
                return {
                    "choices": [{"message": {"content": "模拟响应"}}]
                }  # 模拟 API 响应

        completions = MockCompletions()

    chat = MockChat()


def test_initialization_errors():
    """测试无效的 ChatSeekrFlow 初始化是否引发预期错误。"""

    test_cases = [
        {
            "name": "缺少客户端",
            "args": {"client": None, "model_name": "seekrflow-model"},
            "expected_error": "SeekrFlow 客户端不能为 None。",
        },
        {
            "name": "缺少模型名称",
            "args": {"client": MockSeekrClient(), "model_name": ""},
            "expected_error": "必须提供有效的模型名称。",
        },
    ]

    for test in test_cases:
        try:
            print(f"运行测试：{test['name']}")
            faulty_llm = ChatSeekrFlow(**test["args"])

            # 如果未引发错误，则测试失败
            print(f"❌ 测试 '{test['name']}' 失败：未引发错误！")
        except Exception as e:
            error_msg = str(e)
            assert test["expected_error"] in error_msg, f"意外错误：{error_msg}"
            print(f"✅ 预期错误：{error_msg}")


# 运行测试
test_initialization_errors()
```

```text
运行测试：缺少客户端
✅ 预期错误：SeekrFlow 客户端不能为 None。
运行测试：缺少模型名称
✅ 预期错误：必须提供有效的模型名称。
```
