本笔记本介绍如何开始使用 [Cohere 聊天模型](https://cohere.com/chat)。

前往 [API 参考](https://reference.langchain.com/python/langchain-cohere/chat_models/ChatCohere) 查看所有属性和方法的详细文档。

## 设置

集成位于 `langchain-cohere` 包中。我们可以通过以下命令安装：

```bash
pip install -U langchain-cohere
```

我们还需要获取 [Cohere API 密钥](https://cohere.com/) 并设置 `COHERE_API_KEY` 环境变量：

```python
import getpass
import os

os.environ["COHERE_API_KEY"] = getpass.getpass()
```

设置 [LangSmith](https://smith.langchain.com/) 以获得最佳的观测性体验也很有帮助（但不是必需的）：

```python
os.environ["LANGSMITH_TRACING"] = "true"
os.environ["LANGSMITH_API_KEY"] = getpass.getpass("输入你的 LangSmith API 密钥：")
```

## 使用

ChatCohere 支持所有 [ChatModel](/oss/python/langchain/models) 功能：

```python
from langchain_cohere import ChatCohere
from langchain.messages import HumanMessage
```

```python
chat = ChatCohere()
```

```python
messages = [HumanMessage(content="1"), HumanMessage(content="2 3")]
chat.invoke(messages)
```

## 工具调用

Cohere 支持工具调用功能！

```python
from langchain.messages import (
    HumanMessage,
    ToolMessage,
)
from langchain.tools import tool
```

```python
@tool
def magic_function(number: int) -> int:
    """对整数执行魔法操作
    参数：
        number: 要执行魔法操作的数值
    """
    return number + 10


def invoke_tools(tool_calls, messages):
    for tool_call in tool_calls:
        selected_tool = {"magic_function": magic_function}[tool_call["name"].lower()]
        tool_output = selected_tool.invoke(tool_call["args"])
        messages.append(ToolMessage(tool_output, tool_call_id=tool_call["id"]))
    return messages


tools = [magic_function]
```

```python
llm_with_tools = chat.bind_tools(tools=tools)
messages = [HumanMessage(content="magic_function(2) 的值是多少？")]
```

```python
res = llm_with_tools.invoke(messages)
while res.tool_calls:
    messages.append(res)
    messages = invoke_tools(res.tool_calls, messages)
    res = llm_with_tools.invoke(messages)

res
```
