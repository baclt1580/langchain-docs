## 设置

要访问 ChatYi 模型，您需要创建一个 01.AI 账户，获取 API 密钥，并安装 `langchain_community` 集成包。

### 凭证

前往 [01.AI](https://platform.01.ai) 注册 01.AI 并生成 API 密钥。完成后，请设置 `YI_API_KEY` 环境变量：

```python
import getpass
import os

if "YI_API_KEY" not in os.environ:
    os.environ["YI_API_KEY"] = getpass.getpass("请输入您的 Yi API 密钥：")
```

要启用模型调用的自动追踪，请设置您的 [LangSmith](/langsmith/home) API 密钥：

```python
os.environ["LANGSMITH_API_KEY"] = getpass.getpass("请输入您的 LangSmith API 密钥：")
os.environ["LANGSMITH_TRACING"] = "true"
```

### 安装

LangChain ChatYi 集成位于 `langchain_community` 包中：

```python
pip install -qU langchain_community
```

]

ai_msg = llm.invoke(messages)
