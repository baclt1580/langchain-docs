我们可以使用 [Doctran](https://github.com/psychic-api/doctran) 库提取文档的有用特征，该库利用 OpenAI 的函数调用功能来提取特定元数据。

从文档中提取元数据有助于多种任务，包括：

* **分类：** 将文档分类到不同类别
* **数据挖掘：** 提取可用于数据分析的结构化数据
* **风格转换：** 改变文本的写作方式，使其更贴近预期的用户输入，从而提升向量搜索效果

```python
documents = [Document(page_content=sample_text)]
properties = [
    {
        "name": "category",
        "description": "这封邮件的类型是什么。",
        "type": "string",
        "enum": ["update", "action_item", "customer_feedback", "announcement", "other"],
        "required": True,
    },
    {
        "name": "mentions",
        "description": "这封邮件中提到的所有人名列表。",
        "type": "array",
        "items": {
            "name": "full_name",
            "description": "被提及人的全名。",
            "type": "string",
        },
        "required": True,
    },
    {
        "name": "eli5",
        "description": "像对 5 岁孩子一样解释这封邮件。",
        "type": "string",
        "required": True,
    },
]
property_extractor = DoctranPropertyExtractor(properties=properties)
```

## 输出

从文档中提取属性后，结果将作为一个新文档返回，其元数据中包含提供的属性。

```python
extracted_document = property_extractor.transform_documents(
    documents, properties=properties
)
```

```python
print(json.dumps(extracted_document[0].metadata, indent=2))
```
