[Abso](https://abso.ai/#router) 是一个开源的 LLM 代理，它根据提示复杂度自动在快速模型和慢速模型之间路由请求。它使用各种启发式方法来选择适当的模型。它速度非常快且延迟很低。

## 安装和设置

```bash
pip install langchain-abso
```
