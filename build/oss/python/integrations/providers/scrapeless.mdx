[Scrapeless](https://scrapeless.com) 提供灵活且功能丰富的数据采集服务，支持广泛的参数自定义和多格式导出。

## 安装和设置

<CodeGroup>
```bash pip
pip install langchain-scrapeless
```

```bash uv
uv add langchain-scrapeless
```
</CodeGroup>

您需要设置您的 Scrapeless API 密钥：

```python
import os
os.environ["SCRAPELESS_API_KEY"] = "your-api-key"
```

## 工具

Scrapeless 集成提供了多种工具：

- [ScrapelessDeepSerpGoogleSearchTool](/oss/python/integrations/tools/scrapeless_scraping_api) - 支持跨所有结果类型全面提取 Google SERP 数据。
- [ScrapelessDeepSerpGoogleTrendsTool](/oss/python/integrations/tools/scrapeless_scraping_api) - 从 Google 检索关键词趋势数据，包括随时间变化的流行度、区域兴趣和相关搜索。
- [ScrapelessUniversalScrapingTool](/oss/python/integrations/tools/scrapeless_universal_scraping) - 访问并提取通常屏蔽机器人的 JS 渲染网站的数据。
- [ScrapelessCrawlerCrawlTool](/oss/python/integrations/tools/scrapeless_crawl) - 爬取网站及其链接页面以提取综合数据。
- [ScrapelessCrawlerScrapeTool](/oss/python/integrations/tools/scrapeless_crawl) - 从单个网页提取信息。
