>[Apache 软件基金会（维基百科）](https://en.wikipedia.org/wiki/The_Apache_Software_Foundation)
> 是一个去中心化的开源开发者社区。他们生产的软件根据 Apache 许可证的条款分发，这是一种用于自由和开源软件（FOSS）的宽松
> 开源许可证。Apache 项目的特点是协作、基于共识的开发流程
> 以及开放且务实的软件许可证，也就是说，它
> 允许免费接收软件的开发者以非免费条款重新分发
> 该软件。每个项目都由一个自我选拔的
> 技术专家团队管理，这些专家是该项目的积极贡献者。

## Apache AGE

>[Apache AGE](https://age.apache.org/) 是一个 `PostgreSQL` 扩展，提供
> 图数据库功能。`AGE` 是 `A Graph Extension` 的缩写，灵感来源于 Bitnine 对 `PostgreSQL 10` 的分支 `AgensGraph`，后者是
> 一个多模态数据库。该项目的目标是创建单一
> 存储，能够同时处理关系型和图模型数据，以便用户
> 可以使用标准 ANSI SQL 以及 `openCypher`，即图查询语言。
> `Apache AGE` 存储的数据元素包括节点、连接它们的边，以及
> 节点和边的属性。

有关 [与 Apache AGE 集成](/oss/python/integrations/graphs/apache_age) 的更多信息。

## Apache cassandra

>[Apache Cassandra](https://cassandra.apache.org/) 是一个 NoSQL、面向行的、
> 高度可扩展和高可用的数据库。从 5.0 版本开始，
> 该数据库附带了向量搜索功能。

有关 [与 Apache Cassandra 集成](/oss/python/integrations/providers/cassandra/) 的更多信息。

## Apache doris

>[Apache Doris](https://doris.apache.org/) 是一个用于
> 实时分析的现代数据仓库。它能够以规模化的方式提供极速的实时数据分析。
>
>通常 `Apache Doris` 被归类为 OLAP，并且在 ClickBench——分析型数据库管理系统基准测试中展现了卓越
> 性能。由于它拥有超快的向量化执行引擎，它也可以用作快速的向量数据库。

有关 [与 Apache Doris 集成](/oss/python/integrations/providers/apache_doris/) 的更多信息。

## Apache kafka

>[Apache Kafka](https://github.com/apache/kafka) 是一个分布式消息系统
> 用于发布和订阅记录流。


## Apache spark

>[Apache Spark](https://spark.apache.org/) 是一个用于
> 大规模数据处理的一体化分析引擎。它提供了 Scala、Java、
> Python 和 R 的高级 API，以及支持通用计算图
> 数据分析的优化引擎。它还支持丰富的高级
> 工具集，包括用于 SQL 和数据框的 `Spark SQL`，用于 pandas 工作负载的 `pandas API on Spark`
> 用于 pandas 工作负载，`MLlib` 用于机器学习，
> `GraphX` 用于图处理，以及 `Structured Streaming` 用于流处理。

有关 [与 Apache Spark 集成](/oss/python/integrations/providers/spark) 的更多信息。
