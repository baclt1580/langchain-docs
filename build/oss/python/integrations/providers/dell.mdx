Dell 是一家全球性科技公司，提供包括人工智能解决方案在内的各种硬件、软件和服务。其 AI 产品组合包括专为 AI 工作负载构建的基础设施，其中包括针对 AI 数据管理优化的 Dell PowerScale 存储系统。

## PowerScale

Dell [PowerScale](https://www.dell.com/en-us/shop/powerscale-family/sf/powerscale) 是一款企业级扩展存储系统，托管着行业领先的 OneFS 文件系统，可部署在本地或云端。

### 安装和设置

<CodeGroup>
```bash pip
pip install powerscale-rag-connector
```

```bash uv
uv add powerscale-rag-connector
```
</CodeGroup>
