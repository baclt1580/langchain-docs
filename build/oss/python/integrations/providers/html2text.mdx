>[html2text](https://github.com/Alir3z4/html2text/) 是一个将页面 `HTML` 转换为干净、易读的纯 `ASCII 文本` 的 Python 包。

ASCII 同时也恰好是有效的 `Markdown`（一种文本到 HTML 的格式）。

## 安装和设置

<CodeGroup>
```bash pip
pip install html2text
```

```bash uv
uv add html2text
```
</CodeGroup>

## 文档转换器

查看 [使用示例](/oss/python/integrations/document_transformers/html2text)。

```python
from langchain_community.document_loaders import Html2TextTransformer
```
