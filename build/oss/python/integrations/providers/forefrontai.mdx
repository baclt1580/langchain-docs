> 微调并推理开源文本生成模型


## Installation and setup

访问 [此页面](https://accounts.forefront.ai/sign-in?redirect_url=https%3A%2F%2Fforefront.ai%2Fapp%2Fapi-keys) 获取 `ForefrontAI` API 密钥，并将其设置为环境变量（`FOREFRONTAI_API_KEY`）。

## LLM

查看 [使用示例](/oss/python/integrations/llms/forefrontai)。

```python
from langchain_community.llms import ForefrontAI
```
