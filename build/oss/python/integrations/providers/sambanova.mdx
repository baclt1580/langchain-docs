客户正转向 [SambaNova](https://sambanova.ai/)，以快速部署最先进的 AI 能力，从而获得竞争优势。我们专为企业规模打造的 AI 平台是下一代 AI 计算的技术骨干。我们为基础模型提供动力，这些模型能够解锁数据中蕴藏的宝贵商业洞察。

专为 AI 设计的 SambaNova RDU 采用了革命性的数据流架构构建而成。这种设计使得 RDU 在处理这些工作负载时比 GPU 显著更高效，因为它消除了对内存的冗余调用，而这是 GPU 功能固有的局限性。这种内置效率是使 RDU 能够在更小的占地面积下实现比 GPU 高得多的性能的功能之一。

在我们的架构之上，我们开发了一些平台，允许公司和开发人员充分利用 RDU 处理器和开源模型。


## 安装和设置

安装集成包：

<CodeGroup>
```bash
pip install langchain-sambanova
```

```bash uv
uv add langchain-sambanova
```
</CodeGroup>


## API 密钥
将您的 API 密钥设置为环境变量：

如果您是 SambaCloud 用户，请请求一个 [API 密钥](http://cloud.sambanova.ai/apis?utm_source=langchain&utm_medium=external&utm_campaign=cloud_signup) 并将其设置为环境变量：

```bash
export SAMBANOVA_API_KEY="your-sambacloud-api-key-here"
```

或者如果您是 SambaStack 用户，请将您的基础 URL 和 API 密钥设置为环境变量：

```bash
export SAMBANOVA_API_BASE="your-sambastack-envirronment-base-url-here"
export SAMBANOVA_API_KEY="your-sambastack-api-key-here"
```
