>[Remembrall](https://remembrall.dev/) 是一个为语言模型提供
> 长期记忆、检索增强生成和完整可观测性的平台。

## 安装与设置

要开始使用，[在 Remembrall 平台上使用 GitHub 登录](https://remembrall.dev/login)
并复制您的 [设置页面的 API 密钥](https://remembrall.dev/dashboard/settings)。
