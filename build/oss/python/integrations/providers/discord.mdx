>[Discord](https://discord.com/) 是一个 VoIP 和即时通讯社交平台。用户能够通过语音通话、视频通话、文本消息、媒体和文件在私人聊天或称为“服务器”的社区中进行交流
> 服务器是一组持久性的聊天室和语音频道，可以通过邀请链接访问。

## 安装和设置

<CodeGroup>
```bash pip
pip install pandas
```

```bash uv
uv add pandas
```
</CodeGroup>

按照以下步骤下载您的 `Discord` 数据：

1. 进入您的 **用户设置**
2. 然后前往 **隐私和安全**
3. 前往 **请求我的所有数据** 并点击 **请求数据** 按钮

收到您的数据可能需要 30 天。您将收到一封发送到 Discord 注册地址的电子邮件。该电子邮件将包含一个下载按钮，您可以通过它下载您的个人 Discord 数据。
