[Pebblo](https://www.daxa.ai/pebblo) 使开发人员能够安全地加载和检索数据，从而将其生成式 AI 应用投入部署，而无需担心组织的合规性和安全要求。Pebblo SafeLoader 识别加载数据中发现的语义主题和实体，Pebblo SafeRetriever 对检索到的上下文实施身份和语义控制。结果会在 UI 或 PDF 报告中汇总。

## Pebblo 概述

`Pebblo` 为生成式 AI 应用提供了一种安全的数据加载和检索方式。
它包括：

1. **支持身份感知的安全加载器**，用于加载数据并识别语义主题和实体。
2. **安全检索**，对检索到的上下文实施身份和语义控制。
3. **用户数据报告**，总结加载和检索的数据。
