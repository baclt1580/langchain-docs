>[Spreedly](https://docs.spreedly.com/) 是一项允许您安全存储信用卡并使用它们与任意数量的支付网关和第三方 API 进行交易的服务平台。它通过同时提供卡令牌化/金库服务以及网关和接收器集成服务来实现这一功能。由 Spreedly 令牌化的支付方式存储在 Spreedly，允许您独立存储一张卡，然后根据业务需求将该卡传递给不同的端点。
