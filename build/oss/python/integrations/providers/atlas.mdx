>[Nomic Atlas](https://docs.nomic.ai/index.html) 是一个用于与小规模及互联网规模非结构化数据集进行交互的平台。


## 安装和设置

- 使用 `pip install nomic` 安装 Python 包
- `Nomic` 也包含在 langchains 的 poetry 扩展中 `poetry install -E all`


## 向量存储

查看 [使用示例](/oss/python/integrations/vectorstores/atlas)。

```python
from langchain_community.vectorstores import AtlasDB
```
