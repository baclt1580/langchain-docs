本页面介绍如何在 LangChain 中使用 [Nebula](https://symbl.ai/nebula)、[Symbl.ai](https://symbl.ai/) 的 LLM 生态系统。
它分为两部分：安装与设置，以及特定 Nebula 包装器的参考。

## 安装与设置

- 获取 [Nebula API 密钥](https://info.symbl.ai/Nebula_Private_Beta.html) 并将其设置为环境变量 `NEBULA_API_KEY`
- 请参阅 [Nebula 文档](https://docs.symbl.ai/docs/nebula-llm) 以获取更多详细信息。

### LLM

存在一个 Nebula LLM 包装器，您可以通过以下方式访问
```python
from langchain_community.llms import Nebula
llm = Nebula()
```
