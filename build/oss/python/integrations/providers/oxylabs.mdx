[Oxylabs](https://oxylabs.io/) 是一个市场领先的网络情报收集平台，秉承最高的商业、道德和合规标准，使全球企业能够释放数据驱动的洞察。

[langchain-oxylabs](https://pypi.org/project/langchain-oxylabs/) 实现了工具，使大语言模型（LLM）能够与 Oxylabs Web Scraper API 进行交互。


## 安装和设置

<CodeGroup>
```bash pip
pip install langchain-oxylabs
```

```bash uv
uv add langchain-oxylabs
```
</CodeGroup>

## 工具

有关可用工具的详细信息，请参阅 [Oxylabs 工具文档](/oss/python/integrations/tools/oxylabs/)。
