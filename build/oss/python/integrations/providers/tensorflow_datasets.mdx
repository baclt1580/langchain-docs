>[TensorFlow Datasets](https://www.tensorflow.org/datasets) 是一个现成的数据集集合，
> 可与 TensorFlow 或其他 Python 机器学习框架（如 Jax）配合使用。所有数据集均以
> [tf.data.Datasets](https://www.tensorflow.org/api_docs/python/tf/data/Dataset) 的形式暴露，
> 从而实现易于使用且高性能的输入管道。要开始使用，请参阅
> 该 [指南](https://www.tensorflow.org/datasets/overview) 以及
> 该 [数据集列表](https://www.tensorflow.org/datasets/catalog/overview#all_datasets)。


## 安装和设置

您需要安装 `tensorflow` 和 `tensorflow-datasets` Python 包。

<CodeGroup>
```bash pip
pip install tensorflow
```

```bash uv
uv add tensorflow
```
</CodeGroup>

<CodeGroup>
```bash pip
pip install tensorflow-dataset
```

```bash uv
uv add tensorflow-dataset
```
</CodeGroup>
