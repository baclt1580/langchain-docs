## 安装和设置

GreenNode 集成可以通过 pip 安装：

```python
pip install -qU langchain-greennode
```

### API 密钥

要使用 GreenNode Serverless AI，您需要一个 API 密钥，您可以从 [GreenNode Serverless AI](https://aiplatform.console.greennode.ai/api-keys) 获取。API 密钥可以作为初始化参数 `api_key` 传递，或者设置为环境变量 `GREENNODE_API_KEY`。

```python
import getpass
import os

if not os.getenv("GREENNODE_API_KEY"):
    os.environ["GREENNODE_API_KEY"] = getpass.getpass("Enter your GreenNode API key: ")
```

## 聊天模型

```python
from langchain_greennode import ChatGreenNode

chat = ChatGreenNode(
    model="deepseek-ai/DeepSeek-R1-Distill-Qwen-32B",  # Choose from available models
    temperature=0.6,
    top_p=0.95,
)
```

GreenNode [聊天模型](https://python.langchain.com/docs/integrations/chat/greennode/) 的使用

## 嵌入模型

```python
from langchain_greennode import GreenNodeEmbeddings

# Initialize embeddings
embeddings = GreenNodeEmbeddings(
    model="BAAI/bge-m3"  # Choose from available models
)
```

GreenNode [嵌入模型](https://python.langchain.com/docs/integrations/embeddings/greennode) 的使用

## 重排序

```python
from langchain_greennode import GreenNodeRerank

# Initialize reranker
rerank = GreenNodeRerank(
    model="BAAI/bge-reranker-v2-m3",  # Choose from available models
    top_n=-1,
)
```

GreenNode [重排序模型](https://python.langchain.com/docs/integrations/retrievers/greennode-reranker) 的使用
