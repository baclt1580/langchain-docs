>[AI/ML API](https://aimlapi.com/app/?utm_source=langchain&utm_medium=github&utm_campaign=integration) 提供单一 API 以访问 300+ 托管的基础模型（DeepSeek, Gemini, GPT 等），具备企业级正常运行时间和吞吐量。

## 安装和设置

- 安装 AI/ML API 集成包。

  ```bash
  pip install langchain-aimlapi
  ```

- 在 [aimlapi.com](https://aimlapi.com/app/?utm_source=langchain&utm_medium=github&utm_campaign=integration) 创建账户并生成 API 密钥。
- 通过设置 `AIMLAPI_API_KEY` 环境变量进行身份验证。

```python
import os

os.environ["AIMLAPI_API_KEY"] = "aimlapi_..."
```

## 聊天模型

查看 [使用示例](/oss/python/integrations/chat/aimlapi)。

```python
from langchain_aimlapi import ChatAIMLAPI
```

## 大语言模型

查看 [使用示例](/oss/python/integrations/llms/aimlapi)。
