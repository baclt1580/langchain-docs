> [Docling](https://github.com/DS4SD/docling) 将 PDF、DOCX、PPTX、HTML 和其他格式解析为丰富的统一表示，包括文档布局、表格等，使其适用于生成式 AI 工作流，例如 RAG。
>
> 此集成通过 `DoclingLoader` 文档加载器提供 Docling 的功能。

## 安装和设置

只需从您的包管理器安装 `langchain-docling`，例如 pip：

<CodeGroup>
```bash pip
pip install langchain-docling
```

```bash uv
uv add langchain-docling
```
</CodeGroup>

## 文档加载器

`langchain-docling` 中的 `DoclingLoader` 类无缝地将 Docling 集成到 LangChain 中，使您能够：
- 轻松快速地在使用 LLM 的应用中使用各种文档类型，以及
- 利用 Docling 的丰富表示实现高级的、原生文档溯源。

基本用法如下所示：

```python
from langchain_docling import DoclingLoader

FILE_PATH = ["https://arxiv.org/pdf/2408.09869"]  # Docling Technical Report

loader = DoclingLoader(file_path=FILE_PATH)

docs = loader.load()
```

如需端到端使用示例，请查看 [此示例](/oss/python/integrations/document_loaders/docling)。
