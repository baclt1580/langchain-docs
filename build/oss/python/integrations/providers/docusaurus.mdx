>[Docusaurus](https://docusaurus.io/) 是一个静态站点生成器，提供
> 开箱即用的文档功能。


## 安装和设置


<CodeGroup>
```bash pip
pip install -U beautifulsoup4 lxml
```

```bash uv
uv add beautifulsoup4 lxml
```
</CodeGroup>

## 文档加载器

查看 [使用示例](/oss/python/integrations/document_loaders/docusaurus)。

```python
from langchain_community.document_loaders import DocusaurusLoader
```
