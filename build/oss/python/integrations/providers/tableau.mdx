[Tableau](https://www.tableau.com/) 是一个分析平台，使任何人都能够查看和理解数据。


## 安装和设置

<CodeGroup>
```bash pip
pip install langchain-tableau
```

```bash uv
uv add langchain-tableau
```
</CodeGroup>
