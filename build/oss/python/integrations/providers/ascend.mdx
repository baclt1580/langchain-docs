>[Ascend](https://https://www.hiascend.com/) 是由华为提供的神经网络处理单元

本页面介绍如何使用 Ascend NPU 与 LangChain 配合。

### 安装

使用 torch-npu 进行安装：

<CodeGroup>
```bash pip
pip install torch-npu
```

```bash uv
uv add torch-npu
```
</CodeGroup>

请按照以下指定的安装说明进行操作：
* 按照 [CANN 安装文档](https://www.hiascend.com/document/detail/zh/canncommercial/700/quickstart/quickstart/quickstart_18_0002.html) 所示安装 CANN。

### 嵌入模型

查看 [使用示例](/oss/python/integrations/embeddings/ascend)。

```python
from langchain_community.embeddings import AscendEmbeddings
```
