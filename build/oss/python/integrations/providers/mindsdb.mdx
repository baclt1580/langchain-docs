title: "MindsDB 集成"
description: "使用 LangChain Python 与 MindsDB 集成。"
---

MindsDB 是一个基于企业数据定制 AI 的平台。借助 MindsDB 及其近 200 个到 [数据源](https://docs.mindsdb.com/integrations/data-overview) 和 [AI/ML 框架](https://docs.mindsdb.com/integrations/ai-overview) 的集成，任何开发者都可以利用他们的企业数据来定制 AI 以满足其目的，更快且更安全。

通过 MindsDB，您可以将任何数据源连接到任何 AI/ML 模型，以实现和自动化 AI 驱动的应用程序。利用来自数据库、向量存储或应用程序的数据，实时部署、服务和微调模型。使用开发人员已经熟悉的通用工具完成所有这些操作。

MindsDB 与 LangChain 集成，使用户能够：


- 在 MindsDB 内部部署通过 LangChain 可用的模型，使其可被众多数据源访问。
- 在 MindsDB 内部使用实时和动态数据微调通过 LangChain 可用的模型。
- 使用 LangChain 和 MindsDB 自动化 AI 工作流。

请查阅 [我们的文档](https://docs.mindsdb.com/integrations/ai-engines/langchain)，了解更多关于 MindsDB 与 LangChain 集成的信息并查看示例。
