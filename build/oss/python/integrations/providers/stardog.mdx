[Stardog](https://www.stardog.com) 是一个企业知识图谱平台，使组织能够统一、查询和分析其数据。

为代理提供以下工具：

- 询问自然语言问题并从您的企业知识图谱数据库获取无幻觉的答案
- 从自然语言生成 SPARQL 查询
- 使用语义推理查询和分析结构化数据
- 通过对话界面访问企业数据

### 工作原理
Stardog 提供结合数据集成与自然语言处理的知识图谱基础设施：

1. 让您的代理访问企业知识图谱
2. 通过 Voicebox 使用自然语言查询数据
3. 获取准确、有依据且由您的数据支持的答案
