>[苹果公司（维基百科）](https://en.wikipedia.org/wiki/Apple_Inc.) 是一家美国
> 跨国公司和科技公司。
>
> [iMessage（维基百科）](https://en.wikipedia.org/wiki/IMessage) 是由苹果公司开发并于 2011 年推出的即时
> 消息服务。
> `iMessage` 仅在 Apple 平台上运行。

## 安装和设置

请参阅 [设置说明](/oss/python/integrations/chat_loaders/imessage)。

## 聊天加载器

它从 `iMessage` `chat.db` `SQLite` 文件加载聊天会话。

查看 [使用示例](/oss/python/integrations/chat_loaders/imessage)。

```python
from langchain_community.chat_loaders.imessage import IMessageChatLoader
```
