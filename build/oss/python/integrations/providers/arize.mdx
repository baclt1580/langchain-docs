[Arize](https://arize.com) 是一个 AI 可观测性和大语言模型评估平台，支持 LangChain 应用程序，提供输入、嵌入、检索、函数和输出消息的详细追踪。


## 安装和设置

首先，你需要安装 `arize` Python 包。

<CodeGroup>
```bash pip
pip install arize
```

```bash uv
uv add arize
```
</CodeGroup>

其次，你需要设置你的 [Arize 账户](https://app.arize.com/auth/join) 并获取你的 `API_KEY` 或 `SPACE_KEY`。


## 回调处理器

```python
from langchain_community.callbacks import ArizeCallbackHandler
```
