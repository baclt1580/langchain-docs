[Featherless AI](https://featherless.ai/) 是一个无服务器 AI 推理平台，提供访问超过 4300+ 个开源模型。我们的目标是使所有 AI 模型均可用于无服务器推理。我们通过 API 为持续扩展的开源权重模型库提供推理服务。

# 安装和设置
pip install langchain-featherless-ai
1. 在 [Featherless](https://featherless.ai/register) 注册账户
2. 订阅计划并从 [API Keys](https://featherless.ai/account/api-keys) 获取您的 API 密钥
3. 将您的 API 密钥设置为环境变量（`FEATHERLESSAI_API_KEY`）

# 模型目录
访问我们的模型目录以概览所有模型：https://featherless.ai/models
