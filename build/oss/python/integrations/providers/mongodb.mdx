> [MongoDB](https://www.mongodb.com/) 是一种 NoSQL、面向文档的数据库，支持具有动态模式的类 JSON 文档。

**注意：**
- 参见 [MongoDB Atlas 页面](/oss/python/integrations/providers/mongodb_atlas) 上的其他 `MongoDB` 集成。

## 安装和设置

安装 Python 包：

<CodeGroup>
```bash pip
pip install langchain-mongodb
```

```bash uv
uv add langchain-mongodb
```
</CodeGroup>
