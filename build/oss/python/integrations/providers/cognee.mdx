Cognee 实现了可扩展、模块化的 ECL（提取、认知、加载）管道，允许
您互联和检索过去的对话、文档和音频
转录，同时减少幻觉、开发人员工作量和成本。

Cognee 合并了图数据库和向量数据库，以揭示数据中隐藏的关系和新
模式。您可以自动建模、加载和检索实体和
对象，代表您的业务领域并分析它们之间的关系，揭示
仅靠向量存储或图存储单独无法提供的见解。

在 Google Colab <a href="https://colab.research.google.com/drive/1g-Qnx6l_ecHZi0IOw23rg0qC4TYvEvWZ?usp=sharing">笔记本</a> 中尝试，或查看 <a href="https://docs.cognee.ai">文档</a>。

如果您有问题，请加入 cognee <a href="https://discord.gg/NQPKmU5CCg">Discord</a> 社区。

您见过 cognee 的 <a href="https://github.com/topoteretes/cognee-starter">入门仓库</a> 吗？去看看！


## 安装和设置

<CodeGroup>
```bash pip
pip install langchain-cognee
```

```bash uv
uv add langchain-cognee
```
</CodeGroup>
