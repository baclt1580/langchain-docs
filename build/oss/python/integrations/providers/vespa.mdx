>[Vespa](https://vespa.ai/) 是一个功能齐全的搜索引擎和向量数据库。
> 它支持向量搜索（ANN）、词汇搜索以及结构化数据搜索，所有这些都在同一个查询中完成。

## 安装与设置


<CodeGroup>
```bash pip
pip install pyvespa
```

```bash uv
uv add pyvespa
```
</CodeGroup>
