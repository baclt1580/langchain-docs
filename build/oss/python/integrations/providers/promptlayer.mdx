>[PromptLayer](https://docs.promptlayer.com/introduction) 是一个提示词工程平台。
> 它还帮助进行 LLM 可观测性，以可视化请求、版本化提示词并跟踪使用情况。
>
>虽然 `PromptLayer` 确实有 LLMs 就可以直接与 LangChain 集成（例如
> [`PromptLayerOpenAI`](https://docs.promptlayer.com/languages/langchain)），
> 使用回调是将 `PromptLayer` 与 LangChain 集成的推荐方式。

## 安装和设置

要与 `PromptLayer` 配合使用，我们需要：
- 创建一个 `PromptLayer` 账户
- 创建一个 API token 并将其设置为环境变量（`PROMPTLAYER_API_KEY`）

安装一个 Python 包：

<CodeGroup>
```bash pip
pip install promptlayer
```

```bash uv
uv add promptlayer
```
</CodeGroup>


## 回调

查看 [使用示例](/oss/python/integrations/callbacks/promptlayer)。

```python
import promptlayer  # Don't forget this import!
from langchain.callbacks import PromptLayerCallbackHandler
```


## 大语言模型

查看 [使用示例](/oss/python/integrations/llms/promptlayer_openai)。

```python
from langchain_community.llms import PromptLayerOpenAI
```


## 聊天模型

查看 [使用示例](/oss/python/integrations/chat/promptlayer_chatopenai)。

from langchain_community.chat_models import PromptLayerChatOpenAI
