> [Annoy](https://github.com/spotify/annoy) (`Approximate Nearest Neighbors Oh Yeah`)
> 是一个带有 Python 绑定的 C++ 库，用于在空间中搜索接近给定查询点的点。它还会创建大型只读基于文件的数据结构，这些结构被映射到内存中，以便多个进程可以共享相同的数据。

## 安装和设置

<CodeGroup>
```bash pip
pip install annoy
```

```bash uv
uv add annoy
```
</CodeGroup>


## 向量存储

查看 [使用示例](/oss/python/integrations/vectorstores/annoy)。
