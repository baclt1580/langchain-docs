专为特定提供商设计的中间件。了解更多关于 [中间件](/oss/python/langchain/middleware/overview) 的信息。

| 提供商 | 可用中间件 | 来源 |
|------------|-------------|--------|
| [Anthropic](/oss/python/integrations/middleware/anthropic) | 提示词缓存、bash 工具、文本编辑器、记忆和文件搜索 | [`langchain-ai/langchain`](https://github.com/langchain-ai/langchain/tree/master/libs/partners/anthropic) |
| [AWS](/oss/python/integrations/middleware/aws) | 提示词缓存 | [`langchain-ai/langchain-aws`](https://github.com/langchain-ai/langchain-aws/tree/main/libs/aws) |
| [OpenAI](/oss/python/integrations/middleware/openai) | 内容审核 | [`langchain-ai/langchain`](https://github.com/langchain-ai/langchain/tree/master/libs/partners/openai) |
| [Cisco AI Defense](https://github.com/cisco-ai-defense/ai-defense-langchain-middleware) | 运行时安全检测 | [`cisco-ai-defense/ai-defense-langchain-middleware`](https://github.com/cisco-ai-defense/ai-defense-langchain-middleware) |
