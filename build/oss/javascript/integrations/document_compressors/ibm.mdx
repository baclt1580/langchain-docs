## 设置

要访问 IBM WatsonxAI 模型，您需要创建一个 IBM watsonx.ai 账户，获取 API 密钥或其他类型的凭据，并安装 `@langchain/community` 集成包。

### 凭据

前往 [IBM Cloud](https://cloud.ibm.com/login) 注册 IBM watsonx.ai 并生成 API 密钥，或提供如下所示的其他身份验证形式。

#### IAM 身份验证

```bash
export WATSONX_AI_AUTH_TYPE=iam
export WATSONX_AI_APIKEY=<您的API密钥>
```

#### Bearer 令牌身份验证

```bash
export WATSONX_AI_AUTH_TYPE=bearertoken
export WATSONX_AI_BEARER_TOKEN=<您的Bearer令牌>
```

#### IBM watsonx.ai 软件身份验证

```bash
export WATSONX_AI_AUTH_TYPE=cp4d
export WATSONX_AI_USERNAME=<您的用户名>
export WATSONX_AI_PASSWORD=<您的密码>
export WATSONX_AI_URL=<URL>
```

将这些变量设置到环境变量中后，对象初始化时将自动进行身份验证。

也可以通过将这些值作为参数传递给新实例来完成身份验证。

## IAM 身份验证

```typescript
import { WatsonxLLM } from "@langchain/community/llms/ibm";

const props = {
  version: "YYYY-MM-DD",
  serviceUrl: "<服务URL>",
  projectId: "<项目ID>",
  watsonxAIAuthType: "iam",
  watsonxAIApikey: "<您的API密钥>",
};
const instance = new WatsonxLLM(props);
```

## Bearer 令牌身份验证

```typescript
import { WatsonxLLM } from "@langchain/community/llms/ibm";

const props = {
  version: "YYYY-MM-DD",
  serviceUrl: "<服务URL>",
  projectId: "<项目ID>",
  watsonxAIAuthType: "bearertoken",
  watsonxAIBearerToken: "<您的Bearer令牌>",
};
const instance = new WatsonxLLM(props);
```

### IBM watsonx.ai 软件身份验证

```typescript
import { WatsonxLLM } from "@langchain/community/llms/ibm";

const props = {
  version: "YYYY-MM-DD",
  serviceUrl: "<服务URL>",
  projectId: "<项目ID>",
  watsonxAIAuthType: "cp4d",
  watsonxAIUsername: "<您的用户名>",
  watsonxAIPassword: "<您的密码>",
  watsonxAIUrl: "<URL>",
```

如果您希望从单个查询中获得自动化追踪，还可以通过取消注释以下代码来设置您的 [LangSmith](/langsmith/home) API 密钥：

```typescript
// process.env.LANGSMITH_API_KEY = "<您的API密钥>";
// process.env.LANGSMITH_TRACING = "true";
```

## 实例化

现在我们可以实例化我们的压缩器：

```javascript
import { WatsonxRerank } from "@langchain/community/document_compressors/ibm";

const watsonxRerank = new WatsonxRerank({
  version: "2024-05-31",
  serviceUrl: process.env.WATSONX_AI_SERVICE_URL,
  projectId: process.env.WATSONX_AI_PROJECT_ID,
  model: "cross-encoder/ms-marco-minilm-l-12-v2",
});
```

## 使用

首先，设置一个基本的 RAG 摄取管道，包括嵌入、文本分割器和向量存储。我们将使用它来对所选查询的相关文档进行重排序：

```javascript
import { readFileSync } from "node:fs";
import { MemoryVectorStore } from "@langchain/classic/vectorstores/memory";
import { WatsonxEmbeddings } from "@langchain/community/embeddings/ibm";
import { CharacterTextSplitter } from "@langchain/textsplitters";

const embeddings = new WatsonxEmbeddings({
 version: "YYYY-MM-DD",
 serviceUrl: process.env.API_URL,
 projectId: "<项目ID>",
 spaceId: "<空间ID>",
 model: "ibm/slate-125m-english-rtrvr",
});

const textSplitter = new CharacterTextSplitter({
  chunkSize: 400,
  chunkOverlap: 0,
});

const query = "总统对 Ketanji Brown Jackson 说了什么";
const text = readFileSync("state_of_the_union.txt", "utf8");

const docs = await textSplitter.createDocuments([text]);
const vectorStore = await MemoryVectorStore.fromDocuments(docs, embeddings);
const vectorStoreRetriever = vectorStore.asRetriever();

const result = await vectorStoreRetriever.invoke(query);
console.log(result);
```

将选定的文档传递给重排序器，并接收每个文档的特定分数

```javascript
import { WatsonxRerank } from "@langchain/community/document_compressors/ibm";

const reranker = new WatsonxRerank({
  version: "2024-05-31",
  serviceUrl: process.env.WATSONX_AI_SERVICE_URL,
  projectId: process.env.WATSONX_AI_PROJECT_ID,
  model: "cross-encoder/ms-marco-minilm-l-12-v2",
});
const compressed = await reranker.rerank(result, query);
console.log(compressed);
```
