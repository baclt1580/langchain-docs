## 设置

要访问 Novita AI 模型，您需要创建一个 Novita 账户并获取一个 API 密钥。

### 凭证

前往[此页面](https://novita.ai/settings#key-management?utm_source=github_langchain&utm_medium=github_readme&utm_campaign=link)注册 Novita AI 并生成 API 密钥。完成后，请设置 NOVITA_API_KEY 环境变量：

```bash
export NOVITA_API_KEY="your-api-key"
```

### 安装

LangChain Novita 集成位于 `@langchain-community` 包中：

<CodeGroup>
```bash npm
npm install @langchain/community @langchain/core
```
```bash yarn
yarn add @langchain/community @langchain/core
```
```bash pnpm
pnpm add @langchain/community @langchain/core
```
</CodeGroup>

## 实例化

现在我们可以实例化我们的模型对象并生成聊天补全。立即试用 [Novita AI DeepSeek R1 API 演示](https://novita.ai/models/llm/deepseek-deepseek-r1?utm_source=github_langchain&utm_medium=github_readme&utm_campaign=link)！

```python
import { ChatNovitaAI } from "@langchain/community/chat_models/novita";

const llm = new ChatNovitaAI({
  model: "deepseek/deepseek-r1",
  temperature: 0,
  // 其他参数...
})
```

## 调用

```python
const aiMsg = await llm.invoke([
  {
    role: "system",
    content: "你是一个将英语翻译成法语的助手。请翻译用户的句子。",
  },
  {
    role: "human",
    content: "I love programming."
  },
]);
```

```python
console.log(aiMsg.content)
```
