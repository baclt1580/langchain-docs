LangChain.js 为 Neural Internet 的 Bittensor LLM 模型提供实验性支持。

以下是一个示例：

```typescript
import { NIBittensorLLM } from "@langchain/classic/experimental/llms/bittensor";

const model = new NIBittensorLLM();

const res = await model.invoke(`什么是 Bittensor？`);

console.log({ res });

/*
  {
    res: "\nBittensor 是一个开源协议..."
  }
 */
```
