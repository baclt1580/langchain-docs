以下是按框架组织的常见使用案例教程。

</Card>

</Card>

### LangChain

[LangChain](/oss/javascript/langchain/overview) 的[智能体](/oss/javascript/langchain/agents)实现让简单用例的入门变得轻松。

<Card title="语义搜索" icon="search" href="/oss/javascript/langchain/knowledge-base" horizontal>
  使用 LangChain 组件构建基于 PDF 的语义搜索引擎。
</Card>

<Card title="RAG 智能体" icon="user-search" href="/oss/javascript/langchain/rag" horizontal>
  创建一个检索增强生成（RAG）智能体。
</Card>

<Card title="SQL 智能体" icon="database" href="/oss/javascript/langchain/sql-agent" horizontal>

<Card title="语音智能体" icon="microphone" href="/oss/javascript/langchain/voice-agent" horizontal>

### LangGraph

LangChain 的[智能体](/oss/javascript/langchain/agents)实现使用了 [LangGraph](/oss/javascript/langgraph/overview) 的原语。如果需要更深度的定制，可以直接在 LangGraph 中实现智能体。

<Card title="自定义 RAG 智能体" icon="user-search" href="/oss/javascript/langgraph/agentic-rag" horizontal>
  使用 LangGraph 原语构建 RAG 智能体，实现细粒度控制。
</Card>



### 多智能体

这些教程展示了[多智能体模式](/oss/javascript/langchain/multi-agent)，融合了 LangChain 智能体与 LangGraph 工作流。
