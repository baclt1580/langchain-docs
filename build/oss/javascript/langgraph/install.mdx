sidebarTitle: 安装

要安装基础的 LangGraph 包：



<CodeGroup>
```bash npm
npm install @langchain/langgraph @langchain/core
```

```bash pnpm
pnpm add @langchain/langgraph @langchain/core
```

```bash yarn
yarn add @langchain/langgraph @langchain/core
```

```bash bun
bun add @langchain/langgraph @langchain/core
```
</CodeGroup>


使用 LangGraph 时，通常需要访问大语言模型（LLMs）并定义工具。你可以按自己的方式实现这一点。

一种方法（我们将在文档中使用）是使用 [LangChain](/oss/javascript/langchain/overview)。

安装 LangChain：



<CodeGroup>
```bash npm
npm install langchain
```

```bash pnpm
pnpm add langchain
```

```bash yarn
yarn add langchain
```

```bash bun
bun add langchain
```
</CodeGroup>


如需使用特定 LLM 供应商的包，你需要单独安装它们。

请参考 [集成](/oss/javascript/integrations/providers/overview) 页面获取供应商特定的安装说明。
