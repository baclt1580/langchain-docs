[Agent Server](/langsmith/agent-server) 是一个用于创建和管理基于代理的应用程序的 API 平台。它提供内置的持久化、任务队列，并支持大规模部署、配置和运行助手（代理工作流）。本更新日志记录了 Agent Server 版本的所有显著更新、功能和修复。
